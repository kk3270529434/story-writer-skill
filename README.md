# 西北灵异故事创作技能 — 马老三·口播说书

> 以"马老三"口播讲述为框架，围绕古玩店老板刘一和会看事儿的陈婆，生成西北五省（陕西、甘肃、宁夏、青海、新疆）灵异/民俗/地方大案短篇故事。**兼容任意大模型。先出大纲确认、再生成正文。**

## 简介

v4.0 全面重构——地域聚焦西北五省、固定三人角色体系、双阶段输出（大纲确认→正文生成）、西北方言词库、自动质检脚本。

### 能力范围

- 🔮 精怪仙家、凶宅撞鬼、地方大案、因果报应、民间传说、奇术秘法
- 🏮 地域仅限：陕西、甘肃、宁夏、青海、新疆（回族题材温和创作）
- 🎙️ 固定角色：马老三（口播讲述）、刘一（古玩店老板）、陈婆（会看事儿）
- 🎯 双风格切换：轻松诙谐 / 严肃沉重
- 📏 篇幅：单集短篇(2000-3500字)、上下集(4000-6000字)
- 🔄 双阶段流程：先出大纲 → 人工确认 → 再生成正文

### 核心特色

- **P0 红线 7 条一票否决**：悬念钩子、主角三要素、冲突升级、结局反转、恐怖递增、多层反转+角色灰度、灵异叙事功能
- **强反转强制设计**：双层反转 + 伪解答机制 + ≥6处伏笔
- **西北方言词库**：五省方言词汇 + 使用规则
- **地域知识库**：五省民俗、地方事件、感官锚点
- **固定人设知识库**：三人性格/语言指纹/关系动态/人设崩塌红线
- **自动质检脚本** (`scripts/quality_check.py`)
- **民族内容红线**：回族题材可创作，温和无敏感

## 使用

```
帮我写个陕西西安的凶宅故事，刘一收了一件不干净的古董，陈婆说这东西不能留
```

**流程**：
1. **大纲阶段**：Skill 输出创作卡 → 情节骨架 → 地域考据 → **等待确认**
2. **正文阶段**：确认后 → 马老三闲聊开场 → 分步创作 → 全局终审

```bash
# 自动质检
python scripts/quality_check.py 我的故事.md --strict
```
