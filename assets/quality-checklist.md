# 故事质量核心自查清单

> 更新日志：2026-06-12 | v1.0 | 基于两篇生成故事的系统性缺陷分析提炼

---

## 使用说明

本清单用于两个时机：
1. **构思阶段**（动笔前）：逐项确认设定无先天缺陷
2. **修改阶段**（初稿后）：逐项扫描，标注问题等级

**等级说明**：
- 🔴 P0：不改则故事基本不成立
- 🟡 P1：严重影响阅读体验
- 🟢 P2：精细化提升

---

## 一、故事内核与设定 🔴

### 1.1 核心规则一致性
> "故事前半程和后半程的规则不一致，导致读者出戏。"

- [ ] **规则一句话**：用一句话写下本故事的核心超自然规则（如"电话会预言危险但每次通话都在消耗接听者的寿命"）
- [ ] **规则自检**：逐一检查每个情节节点——是否都遵守了这条规则？有没有某处为制造转折而偷偷改规则？
- [ ] **规则例外必须有代价**：如果需要打破规则，必须在此之前铺垫"打破规则的代价"
- [ ] 反例检测：前半段电话帮人→后半段电话害人，中间无合理解释 → 🔴 违规

### 1.2 反转升级原则
> "用A标签吸引读者，实际内容是B，且B没有比A更精彩。"

- [ ] **反转必须是升级，不能是降级**："斗法"反转成"演戏"=降级；"斗法"反转成"两人其实是死对头但在联手对付第三个更可怕的东西"=升级
- [ ] **标题承诺必须兑现**：标题暗示的精彩程度，正文必须至少达到。不能"标题诈骗"
- [ ] 反例检测：标题"斗法"→内容"两位老太太演戏守封印" → 🔴 违规

### 1.3 怪异现象服务主线
> "出现的怪事只是'为了怪而怪'，不服务于主线设定。"

- [ ] **每个怪异现象必须有"为什么是这个现象"的解释**：水烧开、鸡面壁——这些具体现象与"封印清朝恶魂"有什么逻辑关联？
- [ ] **怪异现象链**：第1个现象→第2个→第3个之间应该有递进关系（越来越危险/越来越接近真相），不能是随机堆砌
- [ ] 反例检测：鸡面壁与"两个老太太守封印"无因果关联 → 🔴 违规

## 二、人物塑造与行为逻辑 🔴

### 2.1 真实反应原则
> "正常人面对超自然事件时的恐惧、犹豫、求证过程被严重压缩。"

- [ ] **第一次接触异常**：主角必须有"否认→困惑→求证→害怕"的完整过程，每个阶段至少一段描写
- [ ] **反应时间合理**：凌晨接死电话→犹豫"十秒"太短。合理时间：至少犹豫1-2分钟，期间写他的内心斗争
- [ ] **"秘密武器化"不可过快**：从"害怕超自然"到"把它当秘密武器利用"，中间必须有至少一个"被迫再次接触→发现好处→贪念萌芽"的过渡事件
- [ ] **"我"测试**：问自己——如果是我亲身遇到这件事，我会怎么做？把那个真实的、可能有点"怂"的反应写出来

### 2.2 情绪层次原则
> "从害怕→好奇→兴奋→认命，中间缺乏合理的过渡事件。"

- [ ] **情绪转变必须有触发事件**：每个情绪阶段至少要有一个具体事件做转折点
- [ ] **倒退允许**：可以在"不怕了"之后又被吓到，再回到"害怕"——真实的人类反应不是线性的
- [ ] 正确示例：害怕(接电话)→困惑(预言应验)→好奇(主动等电话)→依赖(躲避多次危险)→贪念(不愿告诉别人)→恐惧(最后一次电话变了)→崩溃→认命

### 2.3 配角独立性
> "宿管大爷、小陈等角色只在需要时充当'信息播音员'，缺少独立动机和个性。"

- [ ] **每个配角至少有一个不属于主线的欲望/顾虑**：宿管大爷不只是"提供信息的NPC"——他为什么留在学校？他有什么不想说的？
- [ ] **配角拒绝回答至少一次**：配角不能问什么答什么。至少有一次拒绝或转移话题，保持神秘感
- [ ] **配角有缺点**：提供信息的人可以说谎、记错、添油加醋——不一定是100%准确的

## 三、情节结构与节奏 🟡

### 3.1 信息释放控制
> "通过大段对话或主角搜索，一次性把所有谜底倒出，杀死悬念。"

- [ ] **70%原则**：任何解释场景最多给出70%的信息，剩下30%让读者自己拼
- [ ] **禁止百科式独白**：角色不能像念条目一样解释民间法术（反例："梅山法里管这个叫'寄石'……"）
- [ ] **分层揭示**：第一次→暗示（"田埂下头有东西"）；第二次→部分信息（"三十年前封的"）；第三次→大部分真相（"清朝的，害过三个人"）；但永远不100%说透
- [ ] **查资料必须有阻力**：BBS翻旧帖不能太顺利。至少遇到一次删帖/404/关键信息缺失/需要去物理档案室

### 3.2 高潮力度
> "没有真正让人手心出汗的对抗或绝望瞬间。"

- [ ] **高潮必须是"做"出来的，不是"说"出来的**：不能靠对话揭示真相代替高潮
- [ ] **视觉化高潮**：至少有一个可以用画面呈现的对抗时刻（不是两个人站着对话）
- [ ] **绝望感检查**：高潮段主角是否至少有一瞬间感到"我可能过不去了"？如果没有，高潮力度不够
- [ ] 反例检测：两篇故事最高潮都是"对话解释真相" → 🔴 违规

### 3.3 代价原则
> "刘波没付出任何实质性代价就摆脱诅咒；奶奶守封印三十年除了累没有其他损失。"

- [ ] **任何超自然接触必须有代价**：接电话→寿命/记忆/某种不可逆的失去
- [ ] **解决方式不能免费**：说"谢谢"就解除25年诅咒 = 免费。合理代价：刘波必须每年1月13号回到419房间待一夜，永远不能离开那座城市
- [ ] **代价是可见的**：不能只是"他每年那天醒过来"这种软代价。要有读者能感受到的、具体的失去

### 3.4 结局不泄气
> "用一句对话或一个动作解决积攒全程的危机。"

- [ ] **危机解决需要至少"三步"**：不能一蹴而就。发现方法→尝试失败→付出代价→再试→勉强成功或部分成功
- [ ] **反高潮检测**：如果最后危机是用一句对话解决的（"你走吧""谢谢"），→ 🔴 违规

## 四、氛围与悬念技法 🟡

### 4.1 留白规则
> "应该留白让读者自己想象的部分，被作者急于解释清楚。"

- [ ] **"寄石"检查**：任何法术/灵异物品的解释不超过3句。保持神秘感
- [ ] **鬼魂动机不解释**：清朝恶魂为什么害人？可以不解释。越解释越像破案，越不解释越恐怖
- [ ] **"原理"永远不完整**：奶奶可以展示寄石，但不说"梅山法里管这个叫……"——说"你太奶奶留下的，别的别问"

### 4.2 慢恐惧描写
> "对环境的细微异常、逐渐升级的压迫感着墨太少。"

- [ ] **"寂静时刻"强制**：每个恐怖场景前，至少写2-3句环境/感官描写（灯灭了→主角的呼吸声→走廊那头传来的轻微刮擦声→灯又亮了→什么都没变但感觉变了）
- [ ] **事件减速**：不要在2段内塞3件怪事。每件怪事前后给足反应空间
- [ ] **感官描写分配**：视觉40% + 听觉30% + 触觉20% + 嗅觉10%。多数人只写视觉

### 4.3 幽默时机规则
> "在应该紧张恐怖的情节中插入网络梗或自嘲，破坏沉浸感。"

- [ ] **"抖包袱前先让读者怕够"**：搞笑缓冲必须发生在恐怖已经"发酵"之后——读者已经紧张了至少2-3段，然后才允许一个轻快的打断
- [ ] **恐怖段落内禁止网络梗**：鸡面壁后上网查知乎贴吧 = 在恐怖段落内打破沉浸
- [ ] **相声式幽默优先于网络式幽默**：用相声技法（吃了吐、使相、砸挂），不用贴吧知乎段子

## 五、语言与叙事口吻 🟢

### 5.1 口吻内化
> "大量使用'湃哥'式口语，但未内化成自然的叙事节奏。"

- [ ] **口头禅密度控制**："你琢磨琢磨"每篇最多2次；"咱们今天就说这个"每篇1次（仅用于过渡）
- [ ] **用自己的口头禅**：至少有一处是"自己的话"而非模板句式
- [ ] **节奏自检**：朗读全文，找出所有"不自然"的句子——改到读起来顺为止

### 5.2 对话自然度
> "尤其是解释性台词像念百科。"

- [ ] **解释性独白检测**：任何超过2句的连续解释（角色对另一个人说"XXX是……它来源于……它的作用是……"）→ 标记并改写
- [ ] **"人话"测试**：角色在紧张时会说完整句子吗？还是断断续续、用短句？

### 5.3 结尾规则
> "结尾金句式总结用力过猛。"

- [ ] **禁止哲理性金句结尾**："有些电话你可以接……它已经在等了你很多年" → 这是AI最喜欢的结尾模式
- [ ] **结尾三选一**：①开放式留白（不总结）；②一句话收束（不超过15字）；③收场对（已有定场诗体系）
- [ ] **收场对节奏**：END → 1行空 → 收场对 → 不追加任何金句

## 六、细节与真实感 🟢

### 6.1 数字的意义
> "大量出现时间数字但未与情节产生有机呼应。"

- [ ] **每个特殊数字必须有"为什么是这个数字"**：为什么是2:13？如果只是因为"案发时间是2:13"——太机械。让数字本身参与情节。
- [ ] **数字不超过3组**：核心数字1组，辅助数字最多2组。太多=失去冲击力

### 6.2 调查阻力
> "90年代末的大学生能在BBS轻易翻到数年前的详细死亡记录。"

- [ ] **真相获取必须有阻力**：图书馆档案室锁门、关键页面被撕掉、知情人搬家、旧档案丢失……
- [ ] **至少一个"此路不通"**：主角尝试获取信息的途径，至少有一次失败

## 七、快速自检流程（终审前必做）

```
Step 1: 规则一致性    → 读核心规则一句话 → 扫全篇 → 有矛盾吗？
Step 2: 反转质量      → 反转是升级还是降级？
Step 3: 人物反应      → "我"测试：如果是我，会这么做吗？
Step 4: 信息释放      → 找最大段解释独白 → 砍掉30%
Step 5: 高潮力度      → 高潮是"做"还是"说"？
Step 6: 代价          → 主角付出了什么？读者能感受到吗？
Step 7: 结尾          → 有没有AI味金句？有→删
Step 8: 怪异现象链    → 每个怪事都服务主线吗？
Step 9: 去小说化      → 读起来像茶馆聊天还是出版小说？
Step 10: 巧合合理性   → 关键相遇有没有经过曲折过程？
Step 11: 结局风格     → 结尾有没有消解恐惧？
```

## 八、巧合与逻辑去剧本化（v2.5 新增）

> "关键人物出现得太巧（儿子恰好二十年后找到老韩），缺少现实感。"

### 8.1 巧合合理性检查

- [ ] **"恰好"检测**：故事中是否存在以下情况之一？
  - 失散多年的亲人/仇人恰好出现
  - 关键物品恰好被主角捡到
  - 关键信息恰好被某角色完整掌握
- [ ] **如果存在→必须增加至少一个"寻找/求证过程"**：
  - 多次失败（找了几年、跑了好几个地方）
  - 走错路（找到了错误的人、被假线索误导）
  - 通过中介人辗转联系（不是直接找到，而是"朋友的朋友介绍的"）
- [ ] **禁止一次性"恰好找到"**：任何关键信息的获取，至少经历一次失败
- [ ] **降低唯一性**：如果只有一个人知道真相 → 增加冗余（还有半张旧照片/一封残信等佐证）

### 8.2 寻找过程的曲折度模板

```
第一次尝试 → 失败（找错方向/被人拒绝）
第二次尝试 → 找到部分线索（不完整/有误导）
第三次尝试 → 通过中介人/意外发现 → 找到关键信息
```

> 反例：年轻人"查了好几年"一句带过 → 实际没有任何寻找过程的描写。🔴

## 九、结局风格控制（v2.5 新增）

> "结尾交给警察，过于'正能量'、不符合超自然调性。"

### 9.1 结局风格选项（创作前必选其一）

| 风格 | 说明 | 适用 |
|------|------|------|
| **黑暗循环** | 主角解决了当前危机，但发现更大的循环刚开始 | 长篇连载/系列故事 |
| **开放式惊悚** | 事件看似结束，最后一段暗示威胁仍在 | 都市怪谈/撞鬼 |
| **代价型** | 主角付出不可逆代价（失去重要之物/变成了自己讨厌的样子） | 法术/因果报应 |
| **温情诡异** | 有温度但恐怖基调不破（如老韩失踪后河面漂着黑布褂子） | 茶馆风格首选 |
| **冷峻留白** | 不加评论，事件本身的震撼力说话 | 案件改编 |

### 9.2 结局禁止项

- ❌ "交给警察" / "一切恢复正常"（消解恐惧）
- ❌ "后来他过上了幸福的生活"（破坏恐怖基调）
- ❌ "这个故事告诉我们……"（说教）
- ❌ 所有伏笔全部收回（没有余味）

## 十、去小说化检查（v2.5 新增）

> "故事太像小说抄的""读起来不像茶馆聊天"。

### 10.1 小说味 vs 茶馆味

| ❌ 小说味 | ✅ 茶馆味 |
|---------|---------|
| "第一章""第二节"等章节意识 | "咱接着说""说到这儿我得插一句" |
| 纯第三人称客观叙述 | 叙述者随时跳出来评论 |
| "与此同时，在远方……" | "后来才知道，那天晚上……" |
| "他心中涌起一股……" | "他后来跟我说，那感觉就像……" |
| "月光如水，倾泻在……" | "那天晚上月亮挺亮的" |
| 完整的场景描写（时间地点人物环境） | 碎片化的细节（只写跟故事有关的） |
| 所有角色都有来龙去脉 | 配角可以突然出现突然消失 |

### 10.2 自检
- [ ] 全篇有没有超过连续 10 行全是"客观叙述"无叙述者介入？
- [ ] 读一遍——像不像一个人在茶馆里跟你聊天？
- [ ] 有没有出现小说专用结构词（"与此同时""另一方面""镜头转向"）？
- [ ] 框架去掉后，故事本身是否还能成立？（成立=太像小说）
