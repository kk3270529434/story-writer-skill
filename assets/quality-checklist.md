# 故事质量核心自查清单

> 更新日志：2026-06-21 | v2.7 | 第二轮优化：新增剧情新意度校验、反套路指标纳入P0/P1、新意度自评

---

## 使用说明

本清单用于**终审阶段**。先过 P0（一票否决），再过 P1、P2。每项答"是/否"——答"否"必须标注问题位置并说明修改方案。

**通过条件**：P0 全绿 + P1 通过 ≥4 项 + P2 通过 ≥5 项。

---

## P0 级：故事性红线（一票否决 — 全部"是"才能通过）

> 以下 5 项为故事能否成立的根本。任意一项答"否" → 不可进入后续检查，必须修改。

| # | 检查项 | 判定标准 | 是/否 |
|:--:|--------|----------|:----:|
| 1 | **开篇300字内有反常/悬念钩子** | 前300字出现让读者"嗯？"的瞬间。纯背景/铺垫 ≥300字无钩子 → 否 | |
| 2 | **主角三要素齐全** | 同时具备：①表层诉求（具体可落地）②隐藏心结（与结局呼应）③失败代价（具象化、不可逆） | |
| 3 | **存在核心冲突且逐级升级** | 冲突从"疑似反常"→"影响生活"→"生命威胁"逐级放大。无跳级、无后期弱化 | |
| 4 | **结局有反转/余味/因果落点** | 至少含其一：①意料之外的反转（升级而非降级）②细思极恐的余味 ③因果报应的落点。平淡收尾/"交给警察" → 否 | |
| 5 | **恐怖强度逐级递增** | 第一上(轻) < 第二上(中) < 第三上(重)。后段弱于前段 → 否 | |
| 6 | **剧情走向完全可预测** | 全文只有单层真相，读者从头猜到结局？有无至少2层有效反转？ → 否 | |
| 7 | **核心角色有灰度** | 主角/事件人物/灵异主体是否都具备"光明动机"+"黑暗隐秘"？有无纯善/纯恶角色？ → 否 | |

## P1 级：重要整改项（通过 ≥4 项）

> 以下为严重影响阅读体验的问题。答"否"必须说明原因和修改方案。

| # | 检查项 | 判定标准 | 是/否 |
|:--:|--------|----------|:----:|
| 1 | **三上三下情绪曲线完整** | 三级恐怖 + 三次缓冲是否全部到位？每次缓冲是否自然（依附于人物性格，非硬插）？ | |
| 2 | **年代元素与剧情绑定** | 每个年代元素是否都标注了对应的剧情节点？有无孤立堆砌的"背景板"元素？ | |
| 3 | **关键情节无逻辑硬伤** | 核心规则前后一致？时间线无误？角色行为符合人设且动机成立？ | |
| 4 | **配角有独立功能** | 每个配角是否承担了 ≥1 项剧情功能？是否有纯"信息播音员"式工具人？ | |
| 5 | **主角情绪链条完整** | 首次接触异常 → 否认 → 困惑 → 求证 → 害怕 → 面对。有无跳过中间环节？ | |
| 6 | **信息释放有分层** | 真相是否分层揭示（暗示→部分信息→大部分真相）？有无一次性倒出全部谜底？ | |
| 7 | **高潮是"做"出来的** | 高潮能否画成一格漫画？纯对话揭示真相 → 否 | |
| 8 | **危机解决经过"尝试→失败→再试"** | 主角解决危机是否至少 3 步？一步到位（如"说声谢谢"） → 否 | |
| 9 | **反转有前文伏笔支撑** | 每个反转是否对应 ≥3 处前文伏笔？有无无铺垫的"强行反转"？ → 否 | |
| 10 | **灵异元素有核心叙事功能** | 灵异是否承担信息传递/误导、情绪映射、时空连接或真相还原？纯"吓人" → 否 | |
| 11 | **配角有剧情参与度** | 配角是否仅当"信息播音员"？至少1个配角在#03-#04揭晓与核心事件的关联？ → 否 | |
| 12 | **结局有深层落点** | 结局是否跳出了"母爱/亲情"模板？有无人性灰度/时代回响的复杂落点？ → 否 | |

## P2 级：优化调整项（通过 ≥5 项）

> 以下为精细化提升项。答"否"应尽量修改，但不阻塞通过。

| # | 检查项 | 判定标准 | 是/否 |
|:--:|--------|----------|:----:|
| 1 | **0处一级 AI 禁令词** | 搜索 TOP20 禁令词列表，命中数 = 0？ | |
| 2 | **对话占比 30%-40%** | 对话行数/总行数在区间内？各角色话轮长度有明显差异（≥20%）？ | |
| 3 | **Bold 使用合理** | 5-15处？仅用于章节标记/关键信息/揭示性内容？无滥用？ | |
| 4 | **至少3处具体细节** | 有 ≥3 个具体地名/年份/品牌/价格？无泛泛的"某个小镇""一家餐厅"？ | |
| 5 | **口语化电台风格一致** | 朗读全文——有无"读不出来"的段落？有无连续 10 行纯第三人称客观叙述？ | |
| 6 | **无连续500字平铺区** | 有无连续 500 字以上无冲突/无转折/无情绪变化的段落？ | |
| 7 | **至少3处幽默** | 全篇 ≥3 处笑点？用了 ≥3 种不同幽默技巧？ | |
| 8 | **0处对仗式金句结尾** | 结尾是否避免了"有些……有些……而有些……"模式？最后一段是具体画面而非抽象总结？ | |
| 9 | **留白≥30%** | 任何解释场景是否最多给了 70% 信息？民俗解释不超过 3 句？ | |
| 10 | **节尾钩子齐全** | #01 #02 #03 的结尾是否都有钩子（不完全信息/假安全感/冲向悬崖）？ | |
| 11 | **叙事手法不单一** | 是否使用了至少1种非线性叙事手法（不可靠叙述/时空交错/多视角/物品视角等）？全程线性 → 否 | |

## 快速自检流程

```
Step 1: 开篇300字      → 第一个反常/悬念在哪一行？
Step 2: 主角三要素      → 表层诉求？隐藏心结？失败代价？
Step 3: 恐怖递增       → 三级恐怖强度是否 3 < 5 < 8？
Step 4: 结局           → 反转/余味/因果——至少占一个？
Step 5: 三上三下       → 情绪曲线画出来——有没有断裂？
Step 6: AI味扫描       → 搜索 TOP20 禁令词 → 命中数？
Step 7: 朗读一遍       → 录下来自己听 → 哪段不像人说话？
```

## 剧情新意度自评（🆕 v2.7）

> **创作完成后，对标俗套模板给出新意度评分。低于 6 分必须调整剧情。**

### 评分标准

| 分值 | 定义 | 判定 |
|:----:|------|------|
| **1-3** | 完全按俗套模板走，读者看开头就能猜到结局 | ❌ 必须重设核心设定 |
| **4-5** | 大部分走俗套，结尾有一个小反转但不够 | ❌ 需增加至少1层反转 |
| **6-7** | 有1层有效反转，读者在#03-#04被惊到 | ✅ 及格，可提交终审 |
| **8-9** | 有2层以上反转，伪解答设计精妙，读者回头重读发现"原来那时就在暗示" | ✅ 优秀 |
| **10** | 多重反转 + 人性灰度 + 时代落点，读者看完后沉默 | ✅ 极少数 |

### 自评问题

```
1. 读者看到第几节能猜出结局？ [#01就猜到 / #03才猜到 / 看完才知道]
2. 这个故事最让你自己觉得"妙"的反转是哪个？ [写出具体反转内容]
3. 对标 assets/anti-cliche.md 的 TOP20 俗套——你的故事规避了几个？ [计数]
4. 如果用一句话告诉朋友"这个故事其实是在讲什么"——那句话里有几个层次？ [计数]
```

**自评新意度：___ / 10**

**≥6分 → 进入终审。 <6分 → 回到步骤二重新设计反转。**
