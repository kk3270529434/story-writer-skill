## 一、法术/民俗知识库

> 可用于故事中的法术设定、民俗考据、专业术语。

### 1.1 蛊术体系（来源：《苗疆蛊事Ⅱ》）

**核心概念**：以毒虫为媒介的巫术。蛊师通过"养蛊"（让毒虫互噬炼成蛊种）获得能力。

| 类型 | 说明 | 用法提示 |
|------|------|----------|
| 血蛊 | 最恶毒，以鲜血炼制 | 可作故事中的禁术 |
| 情蛊 | 控制情感，下到人身上 | 情感线的超自然驱动 |
| 经血蛊 | 特定蛊种 | 民俗细节素材 |

**规则**：蛊虫需人体宿主培养 → 怕雄黄等药物 → 蛊师间有等级规矩 → 养蛊需持续供养否则反噬。**代价**：折寿、反噬、后代受牵连。

> 原文素材："需要选取十八个具有蛊毒体质、经血旺盛的强壮男性作为炉鼎。"

### 1.2 茅山符箓体系（来源：《茅山守尸人》《捉鬼记》）

**核心法器**：黄符纸、朱砂、桃木剑、铜钱剑、八卦镜、七星灯

**符箓类型**：镇尸符（贴额头镇尸变）、定身符、驱邪符、镇魂符、招魂幡

**关键仪式**：
- 舌血/指尖血画符（紧急情况，威力最大但伤身）
- 焚符化灰追踪邪物
- 净坛 + 开光仪式
- 七星灯续命

> 原文素材："不慌不忙，掏出一张黄符纸，手指一弹，黄符纸便直挺挺地飞出，贴在尸体的额头上，大拇指按住尸体眉心的位置，尸体突然就安静了下来。"

**口诀示例**："天地无极，乾坤借法，五行归一，鬼魅辟易" / "太上赦令，驱邪避凶，急急如律令"

### 1.3 走阴/通阴体系（来源：《走阴人》）

**核心概念**：走阴人可自由穿梭阴阳两界。

**仪式步骤**：血馒头（公鸡血浸泡）→ 撒黑狗血粉画圈 → "引路纸"系脚 → 嚼甘草含片 → 灵魂出窍

**阴间官制**（可用于故事设定）：城隍爷 → 判官 → 无常鬼 → 牛头马面

**规则**：走阴时身体不可受惊扰、不可回头、不可吃阴间食物、必须天亮前返回。

**代价（五弊三缺）**：鳏、寡、孤、独、残（五弊）；缺钱、缺命、缺权（三缺）。

### 1.4 黄河捞尸体系（来源：《黄河捞尸人》）

**行业规矩**：
- 只收男不收女，一师一徒
- 捞尸前必须祭河神（杀公鸡洒血、唱祭词、烧黄纸、撒纸钱）
- 死尸不能用手直接捞 → 用尸钩
- 捞起的尸体要立刻盖脸
- 天黑不捞尸（"暮夜捞尸，阴气太重，容易招惹尸煞"）

> 原文素材：祭词——"早死早超生，黄泉路上好赶路……"

**专用名词**：尸煞、水鬼、河怪、水龙王、巡河使

### 1.5 风水/墓葬体系（来源：《阴墓阳宅》）

**关键概念**：
- 养尸地：特殊地质使尸体不腐，成"僵尸"
- 骑龙葬：高级风水葬法
- 凶地杀坟：位置不当的墓地会害人

**辟邪物**：公鸡血、童子尿、黑狗血、五帝钱、糯米

## 二、恐怖技法库

> 可直接套用的写作技巧。

### 2.1 开篇技法

| 技法 | 来源 | 操作方式 |
|------|------|----------|
| 规则陷阱式 | 地狱电影院 | 主角被不可抗拒的规则卷入（"影院的规则必须严格遵守"） |
| 世代诅咒式 | 鬼咒 | 家族的古老诅咒在当代触发（"每代只有一个男丁"） |
| 职业驱动式 | 法医恐怖事件 | 专业身份自然接触异常尸体（"替死人说话的职业"） |
| 委托/召唤式 | 阴墓阳宅 | 神秘客户上门，把主角拖入事件 |
| 回乡触发式 | 恐怖都市 | 回到偏僻老家/农村，触发沉睡的恐怖 |

**⚠️ 规则**：85%以上的成功作品在**前1500字内**引爆第一个恐怖事件。不可拖沓。

### 2.2 恐怖递进模板

```
Step 1: 环境感官先行（声音/气味/温度变化 → 不写视觉）
Step 2: 主角身体反应（"汗毛竖起来""后背一凉"）
Step 3: 部分视觉（"只能看到腿/背影"→ 不全貌）
Step 4: 恐怖事件发生
Step 5: 事后反应（不可立刻切走，给2-3段反应时间）
```

> 原文范例（你的尸首我的魂）：先看到"两条白得很的长腿" → 再抬头看到"肠子顺着大腿垂下来" →逐步揭示完整恐怖画面。

### 2.3 感官分配黄金比

| 感官 | 占比 | 应用 |
|------|:----:|------|
| 视觉 | 40% | 但延迟使用——先用其他感官 |
| 听觉 | 30% | 脚步声、滴水声、电话铃——最有效的恐怖触发器 |
| 触觉 | 20% | "一股阴风吹来""背后有人推了一把" |
| 嗅觉 | 10% | "霉味""腥臭味"——暗示腐烂和死亡 |

**⚠️ 规则**：最恐怖的永远是"听到但看不到"的东西。让读者先听、再感受、最后才看到。

### 2.4 断章/钩子技巧

| 类型 | 示例 | 来源 |
|------|------|------|
| 数字骤变 | "生存券数目骤然降到57张！" | 地狱电影院 |
| 反预期 | "客车的车门，竟然关上了！" | 地狱电影院 |
| 身份质疑 | "你……你是人还是鬼？" | 多部 |
| 轻微动作 | "她的手好像轻轻地动了一下" | 你的尸首我的魂 |

### 2.5 恐惧中的对话特征

- **短促破碎**："不……不行……我不会就这样。"
- **结巴重复**："你……你……你不是人？"
- **行话/命令**（专家角色）："退后！不许动！"
- **对比张力**：冷静专家 vs 崩溃受害者（"贫道不知道可管得了这桩事" vs "求求你，救救我的孩子！"）

## 三、情节结构库

> 可复用的结构模式。

### 3.1 五种主角原型

| 类型 | 职业 | 优点 | 缺点 | 来源 |
|------|------|------|------|------|
| 厌倦者 | 白领/上班族 | 渴望意义，易被卷入 | 平庸，能力不足 | 无限恐怖 |
| 普通人 | 大学生/打工者 | 读者极易代入 | 一开始不信鬼 | 恐怖都市、古国归墟 |
| 职业者 | 法医/警察 | 专业能力驱动叙事 | 过度理性忽略灵异 | 法医恐怖事件 |
| 传承者 | 家族职业继承人 | 自带知识和法器 | 背负家族诅咒 | 黄河捞尸人 |
| 圈外人 | 销售/闲散人员 | 从零学习，带动读者 | 依赖专家角色 | 阴墓阳宅 |

### 3.2 专家角色引入时机对照

| 时机 | 效果 | 来源 |
|------|------|------|
| **第1页即登场** | 规则集中曝光，快节奏 | 无限恐怖（张杰） |
| **第1个事件中登场** | 主角先独自面对恐怖，再获救 | 阴墓阳宅（叶一） |
| **多章后登场** | 前期悬疑感最强 | 恐怖都市（猫爷，第5章） |
| **电话远程指导** | 神秘感保持最久 | 法医恐怖事件（师父） |

**⚠️ 规则**：专家越早登场，规则说明越集中；越晚登场，前期悬疑感越强。夜间故事建议让专家在第1个事件末尾登场。

### 3.3 危机升级通用路径

```
个体生存 → 连环事件 → 背后更大阴谋 → 威胁扩及他人 → 发现自身与事件的深层关联
```

### 3.4 信息揭示的三种模式

| 模式 | 操作 | 适用 |
|------|------|------|
| **集中披露式** | 专家一次性解释世界观（无限恐怖） | 规则复杂的设定 |
| **碎片拼图式** | 每个案件/章节给一块拼图（法医恐怖） | 悬疑侦探型 |
| **实践教学式** | 主角（和读者）通过亲历事件学习（阴墓阳宅） | 茶馆风格的短篇 |

### 3.5 激励事件时间窗口

| 速度 | 字数 | 效果 |
|:----:|------|------|
| 极快 | < 500字 | 抓人但容易仓促 |
| 快 | 500-1500字 | **推荐**——够铺垫又不拖 |
| 中 | 1500-3000字 | 需要强铺垫的复杂设定 |

## 四、与skill的集成规则

### 4.1 创作时引用

**法术类故事** → 从第一章选取对应法术体系，遵守"规则+代价"框架
**恐怖描写** → 遵循"感官分配黄金比"和"恐怖递进五步模板"
**结构设计** → 从第三章选取主角原型+专家引入时机+信息揭示模式
**开篇选择** → 从第二章"五种开篇技法"中选取一种

### 4.2 禁止事项（强化版）

| ✅ 允许 | ❌ 禁止 |
|--------|--------|
| 使用法术体系的规则（蛊虫互噬、捞尸祭河神等民俗知识） | 搬运小说的人物关系（师徒反目、父子传承、兄弟背叛） |
| 参考恐怖技法（感官分配比、递进五步、断章钩子） | 模仿小说的具体情节结构（"被卷入→遇导师→打BOSS"三幕） |
| 借用单个民俗术语（尸煞、五弊三缺、送河神） | 复制超过2句的小说原文描写 |
| 套用通用写作技巧（开篇技法、信息揭示模式） | 改编角色设定后直接使用（需完全原创角色+关系） |
| 参考"感官分配黄金比"等抽象写作规则 | 把小说情节"换皮"（改名换地保留原结构） |

**核心原则**：novel-reference 提供的是**建筑材料**（砖、瓦、技法），不是**建筑图纸**（情节结构、人物关系）。用砖瓦盖自己的房子——不要把人家的房子重新粉刷一遍。

**去小说化约束（v2.5新增）**：
- 从 novel-reference 中选取的任何法术体系，必须经过"茶馆口吻改写"——去掉所有小说式描写，保留规则骨架，用聊天语气表达
- 不可在故事中出现"小说感"的转场和结构（如"镜头切换""与此同时"）
- 人物的情绪反应不能像小说角色——必须像"一个真实的人跟你讲他遇到的怪事"
