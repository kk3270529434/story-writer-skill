# 经典开头模板库

> 更新日志：2026-06-11 | v1.0 | 基于143篇短篇故事提炼

## 模板一：听众留言引入式（推荐首选）

### 结构
1. 开场白："大家好，我是杨湃。"
2. 引用一个"听众/朋友"的留言或对话
3. 对此发表个人看法（夹叙夹议）
4. "咱们今天就给大伙讲一个……"过渡
5. `**#01**` 正式进入

### 范文示例
```
大家好，我是杨湃。

最近评论区有一位朋友叫无邪给我留言，说经常听人念叨我们东北五大家仙，什么胡黄白柳灰，胡黄白柳还算常见，但是怎么没听谁讲过灰仙的故事，希望主播有机会给大伙讲讲。

灰仙儿，这指的是老鼠，北方人叫耗子。一般供这个，求财的居多。你像灰仙独有的一个本领叫运财之术，也就是老百姓常说的"搬运"。但是因为与生俱来的身份不是很光彩，耗子嘛，老鼠过街人人喊打，一般人供了也不往外说，所以市面上关于灰仙的故事也就很少，但并不代表没有。

咱们今天就给大伙讲一个灰仙的故事。

**#01**
```

### 适用场景
- 有明确主题/关键词的故事
- 需要建立"真实感"的故事
- 故事与某种特定民间信仰/习俗相关

## 模板二：朋友经历引入式

### 结构
1. "大家好，我是杨湃。"
2. 直接叙述"我有一个朋友/听众/熟人"的某段经历
3. 交代时空背景（哪年、在哪、什么情况）
4. 自然过渡到故事核心

### 范文示例
```
大家好，我是杨湃。

今天要讲的故事，是我上次直播时一位名叫包子有肉的朋友，跟我们分享的一则发生在他大姨夫身上的真实经历。

故事发生在1983年，大姨夫顾名思义就是大姨的丈夫，但包子他妈在这帮兄弟姐妹里排行最小，所以83年那会儿大姨已经结婚了，咱这老姨却还上中学。

**#01**
```

### 适用场景
- 第一人称亲历型故事
- 需要多层人物关系的"转述型"叙事
- 农村/小镇背景的诡异事件

## 模板三：设问引入式

### 结构
1. "大家好，我是杨湃。"
2. 抛出一个引人深思的问题
3. 给出自己的看法/讨论
4. 用具体故事来回答

### 范文示例
```
大家好，我是杨湃。

有听友曾经问过我这样一个问题："老看电影电视剧里头把医院太平房，那也就是停尸间来设定成恐怖故事的发生地。而咱们平时有个头疼脑热的也上医院，怎么就从来没看过这停尸间在哪呢？"

咱们今天就给大伙讲一个医院太平房的故事。

**#01**
```

### 适用场景
- 日常生活相关的都市传说
- 需要打破常识/颠覆认知的故事
- 听众容易产生共鸣的话题

## 模板四：年代锚定式（历史/民间传说专用）

### 结构
1. "大家好，我是杨湃。"
2. 可选：简短铺垫（1-2行）
3. 直接用年代+地点+事件开篇
4. `**#01**` 呼应时间线

### 范文示例
```
大家好，我是杨湃。

从现在开始就关注我的老朋友们应该都还记得，深夜小茶馆的第一个故事，说的是上海龙柱事件。当时为什么选择这个内容？因为我自己心里头有点小算盘——咱们中国人对待龙这个东西，全都有着一种特殊的崇拜。

那么今天呢，咱们就再来讲一个跟龙有关的故事。

**#01**

1960年，当时由于一些特别情况，对于生产力的发展进行了过度乐观的错误判断，提出了一个口号叫："三年时间赶超英美，举国上下大炼钢铁。"
```

### 适用场景
- 历史背景强相关的故事
- 大跃进/文革/改革开放等特定年代
- 需要大量历史context铺垫的民间传闻

## 模板五：人物先行式

### 结构
1. "大家好，我是杨湃。"
2. 先介绍主角（姓名、年龄、现状）
3. 用人物引出故事

### 范文示例
```
大家好，我是杨湃。

这人叫老马，今年50来岁。干什么的不知道，反正隔三岔五就来我这聊聊天，今天弄个成化的茶碗，明天带个宣德的香炉，美其名曰交流研讨，但实际上就为蹭我茶喝，没办法谁让我好这个呢。

沏了壶茶，老马才说："我年轻那会儿还真挺能干的，但是呢，干什么赔什么……"
```

### 适用场景
- 茶馆常客/熟人的故事
- 角色驱动的叙事
- 需要先建立人物认同再讲事件的类型

## 选择指南

| 故事类型 | 推荐模板 |
|----------|----------|
| 听众点播/互动类 | 模板一：听众留言引入式 |
| 真实经历/亲历类 | 模板二：朋友经历引入式 |
| 都市传说/颠覆认知 | 模板三：设问引入式 |
| 历史事件/民间传说 | 模板四：年代锚定式 |
| 茶馆常客/人物类 | 模板五：人物先行式 |
