# 经典开头模板库

> 更新日志：2026-06-21 | v2.6 | 全量优化：扩容为3类10种、强制300字入题、去同质化

---

## 使用说明

创作时从以下 3 类 10 种模板中选择一种。**所有模板强制满足"300字内进入第一个反常点"**——删减冗长的问候铺垫，快速入题。

---

## 第一类：悬念式开头（3种）

> 适合：细思极恐型、撞鬼/凶宅、都市怪谈。特点：直接抛出怪事，先吓人再展开。

### 模板 1：异常物件式
**结构**：场景锚定 → 物件异常 → 主角反应 → 进入故事

```
大家好，我是杨湃。

[1-2句场景锚定]那天晚上[时间]，[主角]正在[日常动作]。

[异常出现]就在这时候，[物件]突然[异常行为]。[描写细节，2-3句]

[主角反应]他愣了一下。[口头反应]。

**#01**
```

**300字钩子位置**：异常出现（通常在 150-200 字处）

**适配**：撞鬼/凶宅、都市怪谈 | 单集短篇 | 细思极恐调性

### 模板 2：人物失踪式
**结构**：先报结果 → 倒叙回最开始

```
大家好，我是杨湃。

[人物]消失了。那是[时间]的事。到现在也没人知道[核心谜团]。

这事得从[起点时间]说起。

**#01**
```

**300字钩子位置**：第一句话就是钩子

**适配**：任何类型 | 单集/上下集 | 悬疑驱动

### 模板 3：规则宣告式
**结构**：宣告一条诡异规则 → 展示这条规则如何应验

```
大家好，我是杨湃。

[地点]有个规矩——[具体规则]。[不信邪的后果简述]。

[主角]一开始也不信。

**#01**
```

**300字钩子位置**：规则本身即是悬念（约 100 字处）

**适配**：民间传说、因果报应 | 单集短篇 | 冷峻/悬疑调性

## 第二类：生活化开头（4种）

> 适合：民间传说、保家仙、因果报应类。特点：从日常场景切入，逐步反常，读者跟随主角一起发现不对劲。

### 模板 4：朋友经历式（推荐首选）
**结构**：茶馆闲聊 → 引出某人 → 直接进入他的故事

```
大家好，我是杨湃。

[人名]是我[关系]，[年龄/职业/一两句人物速写]。前两天[他来我这/打电话/发消息]，说了件事——

[直接进入他的经历，2-3句即到第一个反常]

**#01**
```

**300字钩子位置**：经历开头即有反常（约 100-150 字处）

**适配**：精怪仙家、因果报应、民间传说 | 单集短篇 | 亲切/茶馆调性

### 模板 5：日常渐变式
**结构**：描写极其普通的日常 → 一个小细节不对劲 → 慢慢放大

```
大家好，我是杨湃。

[主角]的生活一直很规律。[2句日常描写]。

但[具体日期/时间]那天，[一件极小的事]跟平时不太一样。

[主角]没太在意。

**#01**
```

**300字钩子位置**：小细节出现（约 200-250 字处）

**适配**：都市怪谈、细思极恐 | 单集短篇 | "温水煮青蛙"式恐怖

### 模板 6：年代记忆式
**结构**：年代锚定 → 社会背景 1-2 句 → 具体事件

```
大家好，我是杨湃。

[年份]年，[地点]。[1-2句年代特征——通过具体物价/流行/社会事件呈现，非百科式罗列]。

就在那一年，[主角]遇到了这辈子都忘不了的一件事。

**#01**
```

**300字钩子位置**："遇到了这辈子都忘不了的事"即钩子（约 200 字处）

**适配**：民间传说、历史事件 | 上下集/长篇 | 厚重史诗感

### 模板 7：物品来历式
**结构**：展示一件有故事的物品 → 追问来历 → 引出背后的故事

```
大家好，我是杨湃。

[主角]家里有[一件物品]。[外观描写1-2句]。

但这东西的来历——[主角]从来没跟人说过。直到[触发条件]。

**#01**
```

**300字钩子位置**："从来没跟人说过"即钩子（约 100 字处）

**适配**：因果报应、奇术秘法 | 单集短篇 | 倒叙解谜

## 第三类：互动式开头（3种）

> 适合：电台文稿格式、需要建立"真实感"的故事。特点：主播直接与听众对话，引入主题。

### 模板 8：听众点播式
**结构**：引用听众留言 → 个人看法 → 过渡到故事

```
大家好，我是杨湃。

最近有朋友[网名]给我留言，说[具体问题/请求]。

[个人看法，3-5行——但必须与后面的故事主题直接相关，不做无效闲聊]

咱们今天就给大伙讲一个[故事主题]。

**#01**
```

**300字钩子位置**：听众提问本身暗含悬念（约 50 字处）

**适配**：所有类型 | 单集短篇 | 亲切/互动调性

### 模板 9：问题引导式
**结构**：抛出一个问题 → 简短讨论 → 用故事作答

```
大家好，我是杨湃。

你有没有想过——[一个与灵异/民俗有关的问题]？

[2-3句简短讨论，带出一个真实案例/民间说法]

咱们今天就聊聊这个。

**#01**
```

**300字钩子位置**：问题本身即是钩子（第一句）

**适配**：都市怪谈、民间传说 | 单集短篇 | 对话感强烈

### 模板 10：系列承接式
**结构**：简短回顾上一期 → 引入本期主题

```
大家好，我是杨湃。

上回咱们讲了[上期故事一句话概括]。没听过的朋友建议先去补一下，链接在评论区。

今天要说的，是[本期主题]。

**#01**
```

**300字钩子位置**：#01 开始后即入题（本期主题承诺即钩子）

**适配**：连载/系列故事 | 多集 | 承接过渡

## 模板选择指南

| 故事调性 | 推荐模板 | 入题速度 |
|----------|----------|:--------:|
| 细思极恐 | 模板1 异常物件式 / 模板5 日常渐变式 | 最快 |
| 亲切茶馆 | 模板4 朋友经历式 / 模板8 听众点播式 | 中等 |
| 悬疑驱动 | 模板2 人物失踪式 / 模板3 规则宣告式 | 最快 |
| 史诗厚重 | 模板6 年代记忆式 | 较慢 |
| 倒叙解谜 | 模板7 物品来历式 | 中等 |
| 互动对话 | 模板9 问题引导式 | 最快 |

## 强制规则

- ✅ 任何模板，**300 字内必须出现第一个反常/悬念/谜团**
- ✅ "大家好，我是杨湃"为固定开场白（如用户自定义则用自定义）
- ❌ 禁止超过 200 字的纯问候/闲聊/无悬念铺垫
- ❌ 禁止用"在遥远的""在很久以前""众所周知"开头
