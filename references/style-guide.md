# 创作风格手册 — "深夜小茶馆"杨湃风格

> 更新日志：2026-06-21 | v2.7 | 第二轮优化：灵异元素叙事功能分类、细节恐怖规则、情绪曲线适配反转

---

## 一、叙事风格表层

### 1.1 叙事人称
- **第一人称叙事者为默认框架**。所有故事由"杨湃"以茶馆老板身份讲述。
- 故事主体用第三人称展开，但第一人称框架随时介入：
  - 插入个人评论
  - 打断叙事解释民俗/年代背景
- 视角切换用空行分隔，不做特殊标记。
- **禁止**：纯第三人称无框架叙事；学术化旁白。

### 1.2 时态
- 全部使用**过去时**叙述（"那天""这会儿""后来"）
- 框架部分使用**现在时**（"咱们今天给大伙讲一个……"）
- 禁止将来时预告（"接下来我们将看到……"）

### 1.3 开头模式
| 模式 | 占比 | 特征 |
|------|:----:|------|
| 听众互动式 | 60% | 引述听众留言/评论，再过渡 |
| 个人经历式 | 25% | 讲自己的见闻或朋友的故事 |
| 设问引入式 | 10% | 先抛问题，再讲故事 |
| 直入正题式 | 5% | 无铺垫直接开始 |

### 1.4 结尾模式
| 模式 | 占比 | 特征 |
|------|:----:|------|
| 标准收束 | 80% | `END` → 奇闻说今古·兮香话鬼狐 |
| 哲理升华 | 12% | 结尾加一段人生感悟/道理 |
| 开放留白 | 5% | 不加评论，戛然而止 |
| 幽默翻转 | 3% | 讲完恐怖故事后突然搞笑 |

## 二、三上三下情绪曲线（P0 强制 — 具象化三级标准）

> 每篇故事必须经历三级恐怖递增 + 三次缓冲回落。这是不可跳过的叙事骨架。

### 第一轮：轻度恐怖 → 生活化缓冲

| | 第一上（轻度恐怖） | 第一下（生活化缓冲） |
|------|------|------|
| **位置** | #01 中后段 | #01 尾 → #02 开头 |
| **触发内容** | 初遇反常：听到异响、看到残影、物品异位、收到不该收到的消息 | 主角自我安慰："应该是听错了""明天白天再看看" |
| **情绪等级** | 2-3/10 | 回到 1-2/10 |
| **写法要求** | 用环境感官先行（声音/气味/温度），再写视觉异常 | 朋友插科打诨、主角用日常逻辑消解恐惧、叙述者插入吐槽 |
| **禁止** | ❌ 在此就揭示异常的全部面貌 | ❌ 缓冲后紧张感完全消失（要留一根"刺"） |
| **示例** | "走廊的声控灯灭了。刘波听见自己的呼吸声。他跺了一脚——灯亮了。那部电话安安静静地挂在那儿。" | "'电话线都没插，怎么可能响。肯定是隔壁房间的。'刘波把被子蒙在头上，翻了个身。" |

### 第二轮：中度恐怖 → 黑色幽默缓冲

| | 第二上（中度恐怖） | 第二下（黑色幽默） |
|------|------|------|
| **位置** | #02 中段 | #02 尾 → #03 开头 |
| **触发内容** | 直面怪事：亲眼目击异象、被纠缠跟踪、预言成真、求助被拒 | 高人嘴贫吐槽、角色闹乌龙、荒诞细节被放大 |
| **情绪等级** | 5-7/10 | 回到 3-4/10 |
| **写法要求** | 主角开始主动求证但遇阻；每次发现让情况更复杂而非更清晰 | 用人物性格带出笑点（不是跳出叙事讲笑话）；笑点依附于具体角色的言行 |
| **禁止** | ❌ 查资料一帆风顺；❌ 知情人问什么答什么 | ❌ 在恐怖还未"发酵"时就搞笑；❌ 网络梗/贴吧知乎段子 |
| **示例** | "电话里的预言第三个也应验了。刘波开始主动等那通电话。" | "'你说这鬼也是，大半夜的不睡觉，在这打电话——它不用交话费啊？' '你能不能正经点。' '我很正经。我是真的好奇。'" |

### 第三轮：重度恐怖 → 情绪落地

| | 第三上（重度恐怖） | 第三下（情绪落地） |
|------|------|------|
| **位置** | #03 尾 → #04 | #04 尾 → 收束 → END |
| **触发内容** | 生死关头：退路被切断、最终对抗、真相大白 | 温情释然/因果收尾/余味留白 |
| **情绪等级** | 7-9/10 | 回落到 3-4/10 |
| **写法要求** | 主角面临"我可能过不去了"的绝望；真相揭示必须"做"出来而非"说"出来 | 用结局落点平复情绪——具体画面、因果闭环、或留白暗示。**禁止在此阶段再插入搞笑** |
| **禁止** | ❌ 对话揭示全部真相；❌ "说声谢谢"就解决终极危机 | ❌ AI 味金句总结；❌ 硬插搞笑破坏余味 |
| **示例** | "电话那头说——'一月十三号。凌晨两点十三分。419。刘波。'他手里的听筒差点掉在地上。" | "从那以后，电话再也没有响过。但刘波每次路过那层楼，都会走快几步。END" |

### 恐怖递增原则
```
第一上 (2-3/10) < 第二上 (5-7/10) < 第三上 (7-9/10)
```
后段恐怖感不能弱于前段。如果 #04 的恐怖强度不如 #02 → 违规。

### 幽默衔接规则
- ✅ 笑点依附于**人物性格**和**剧情场景**（让角色自己搞笑，不让叙述者跳出故事讲笑话）
- ✅ 幽默发生在恐怖已"发酵"之后（至少 2-3 段紧张描写 → 才允许一个轻快打断）
- ❌ 禁止在恐怖段落进行中插入搞笑（读者还没怕够就笑了 = 恐怖白写了）
- ❌ 禁止跳出叙事强行讲笑话（"说到这儿我想起一个段子……"）

## 三、语言指纹定量分析

### 3.1 句子长度
- **常规叙述**：25-40字/句
- **紧张/行动**：10-20字/句，短促断裂
- **关键转折**：50-80字/句复合句，营造粘滞氛围
- **极端短句**（4-8字）：用于强调和节奏断裂

### 3.2 段落规模
- 平均段落：**1-5行**，极短（口语节奏）
- 单句成段：频繁使用
- 最长段落：不超过 8-10 行，仅用于历史背景阐述

### 3.3 对话占比
- **对话行数 / 总行数 ≈ 30%-40%**
- 对话内嵌于叙述中，不独立分行
- 角色对话用冒号引出
- 禁止纯对话段落连续 5 个话轮以上无叙述穿插

### 3.4 描写面积比
| 类型 | 占比 |
|------|:----:|
| 动作/事件叙述 | 55% |
| 对话 | 35% |
| 环境描写 | 5% |
| 心理描写 | 5% |

### 3.5 高频修辞手法
1. **排比断句**（最高频）："不是缺胳膊就是少腿，还有身上挂着霜正滴滴答答往下滴水的"
2. **设问自答**："那么说这医院为什么会平白无故闹起了诈尸？原来……"
3. **口语化比喻**："给那羊薅的跟葛优似的"
4. **夸张制造幽默**："这一宿，医院改奥运会场地了"
5. **顶针/递进**："他着急。他害怕。他跑。"

### 3.6 口语标签库
- 开头/过渡：那么说、咱单说、说这会儿、这天、列位
- 转折：可谁承想、结果、没成想、但巧了
- 揭秘：原来、敢情是、说白了
- 收束：就这么着、这真是
- 共鸣：你琢磨琢磨、你像
- 惊叹：好家伙、我的妈啊

### 3.7 方言语气词
嘛、呐、啊、呗、咋、啥、甭、咱、忒、咋滴

## 四、句子段落节奏规则

### 4.1 节奏变化表
| 场景类型 | 句长 | 段长 | 对话密度 |
|----------|------|------|:--------:|
| 铺垫/背景 | 30-50字 | 3-5行 | <10% |
| 紧张/追逐 | 10-20字 | 1-2行 | 20% |
| 对话交锋 | 15-25字 | 1-3行 | >50% |
| 揭秘/解释 | 30-60字 | 4-8行 | <5% |
| 幽默段落 | 20-35字 | 2-3行 | 30% |

### 4.2 Bold 使用规则
**必须 Bold**：章节标记 `**#01**`、关键人名地名首次出现、揭示性/反转信息、收场对

**禁止 Bold**：普通强调、超过 50 字的连续段落、对话中所有内容

### 4.3 开篇节奏模板
```
标题（1行）
==== 分隔线
空行
大家好，我是杨湃。
空行
**有话要说** 或直接铺垫（3-15行）
**#01**
```

## 五、幽默技巧库：相声评书体系

> 深夜小茶馆的幽默不是网络段子——是**相声的包袱逻辑 + 评书的扣子功夫**。强制要求：每篇 ≥3 种技巧、≥3 处笑点。

### 5.1 九大笑技法

#### 砸挂（自嘲/调侃熟人）
- **自砸**：杨湃自嘲身份/收入/智商
- **砸熟人**：调侃常来茶馆的角色
- **砸现象**：调侃社会现象（不可过度政治化）
- **禁用对象**：受害者、死者、家属、听众

#### 三番四抖（相声包袱结构）
- 三次铺垫把读者预期往一个方向引 → 第四次突然转向抖开
- 每篇至少 1 处完整结构

#### 吃了吐（出尔反尔）
- 先说大话/表态，立刻自己推翻
- "说到这儿我乐了……这段掐了别播。反正你们也听到了。"

#### 现挂（即兴评论）
- 叙述者突然对"此刻正在发生的事"发表评论
- 要有"刚想到的"即时感

#### 定场诗 + 收场对
- 收场对在 END 之后：`> **奇闻说今古·兮香话鬼狐。下期我们不见不散。**`

#### 扣子（悬念停留）
- 在不完全信息处停下
- #01 尾：展示异常不解释 / #02 尾：假安全感 / #03 尾：冲向悬崖

#### 批讲（暂停故事解释背景）
- 口吻：老辈给晚辈讲古
- 长度：1-5 句

#### 使相（动作画面制造喜剧）
- 用具体动作代替抽象情绪
- "差点没给大奎整吐了"

#### 贯口（快节奏罗列）
- ≥3 项罗列，由慢到快，落地一句反转

### 5.2 技法分布要求
| 故事位置 | 必须技法 |
|----------|----------|
| 框架开场 | 定场诗/词 + 砸挂 |
| "有话要说"过渡 | 批讲 |
| #01 结尾 | 扣子（不完全信息型） |
| 最恐怖处之后 | 吃了吐 或 使相 |
| #03 结尾 | 扣子（冲向悬崖型） |
| 全篇随机 | 现挂(2-3处) + 三番四抖(≥1处) |
| 收尾 | 收场对 |

## 六、留白、慢恐惧与氛围控制

### 6.1 70% 留白原则
任何解释场景最多给出 70% 的信息，剩下 30% 让读者自己拼。民俗解释不超过 3 句。

### 6.2 "寂静时刻"模板
```
[环境感官 2-3句] → [主角身体反应 1-2句] → [细微异常 1-2句] → [恐怖事件发生]
```

### 6.3 感官分配
- 视觉 40% | 听觉 30% | 触觉 20% | 嗅觉 10%

### 6.4 结尾规则
禁止 AI 味金句总结。结尾三选一：开放式留白 / 一句话收束 / 收场对。

## 七、去小说化规则

| ❌ 小说味 | ✅ 茶馆味 |
|---------|---------|
| "与此同时，在远方……" | "后来才知道，那天晚上……" |
| "月光如水，倾泻在……" | "那天晚上月亮挺亮的" |
| "镜头切换到……""画面一转……" | 空一行。下一段直接说 |
| 纯第三人称连续叙述超过 10 行 | 每 5-8 行插入批讲/砸挂 |
| 所有角色都有来龙去脉 | 配角可以突然出现突然消失 |

### 去小说化三步
1. 搜索连续 10 行以上纯客观叙述 → 插入批讲或砸挂
2. 搜索"与此同时""另一方面""镜头切换"→ 替换为口语转场
3. 朗读全文 → "播音腔"的段落重写为"聊天腔"

## 八、灵异元素叙事功能分类（P0 — 禁止无意义灵异描写）

> **每个灵异元素必须标注其核心叙事功能。禁止"只是为了吓人"的灵异描写。**

| 功能类型 | 说明 | 可用场景 | 服务于反转的方式 |
|----------|------|----------|----------------|
| **信息传递/误导** | 传递真相或谎言，可误导也可引导主角 | 电话传话、手绢字迹、收音机广播 | 传递的信息可能是假的——是谁在"发信号"？ |
| **情绪映射** | 对应人物的内心状态，灵异强度=心理压力 | 电话响的频率=人物内疚程度 | 灵异的变化暴露了人物的心理变化 |
| **时空连接** | 连接不同年代，推动因果 | 过去的电话接通现在 | 两个时代的真相互为因果 |
| **真相还原** | 通过灵异场景重现当年真实事件 | 幻象/噩梦/反复出现的场景 | 还原的"真相"与当事人的口述矛盾 |

**自检**：删掉这个灵异元素，反转还能成立吗？能→灵异元素是多余的，需重新设计。

## 九、细节恐怖替代直白恐怖规则（P2 — 新增）

> **减少直白的"鬼现身"，多用物品细节、逻辑矛盾制造细思极恐感。**

| ❌ 直白恐怖（少用） | ✅ 细节恐怖（多用） |
|-----------------|-----------------|
| 鬼影出现、面目狰狞 | 物品位置变了、多了不该有的东西 |
| 角色大喊"有鬼！" | 角色发现一个逻辑上无法解释的矛盾 |
| "一股阴风吹来" | 手绢上的线是新的——但手绢是1968年的 |
| "他感到毛骨悚然" | 他把手电筒的光照在坑壁上——那个位置不该有血迹 |

**核心**：最恐怖的永远不是"看到了可怕的东西"，而是"看到了不该出现的东西"。

## 十、三上三下适配多层反转（升级版）

> 在原三级恐怖基础上，将反转融入情绪曲线。

| 阶段 | 原版 | v2.7 升级版 |
|------|------|-----------|
| **第一上** | 轻度恐怖：初遇反常 | 轻度恐怖 + **埋下表层假象**（引导俗套预判）+ 伏笔1 |
| **第一下** | 生活化缓冲 | 生活化缓冲 + **暗藏第一个矛盾细节**（读者当时察觉不到） |
| **第二上** | 中度恐怖：直面怪事 | 中度恐怖 + **抛出伪解答**（"原来是这样"）+ 伏笔2 |
| **第二下** | 黑色幽默 | 黑色幽默 + **伪解答下的疑点**（笑声背后的一丝不安） |
| **第三上** | 重度恐怖：生死关头 | 重度恐怖 + **伪解答崩塌 → 深层真相揭晓** |
| **第三下** | 情绪落地 | **复杂回响**（不单纯温情，不单纯恐怖——人性+时代的余味） |
