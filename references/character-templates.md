# 人物生成模板

> 更新日志：2026-06-26 | v1.0 | 新建：长线阻碍/心理锚点/善恶反差三类核心模板

---

## 使用说明

在**步骤二（情节骨架）**阶段，为每个主要角色套用对应模板。模板不是填空题——是校验工具。确保每个角色的底层驱动完整。

---

## 模板一：长线角色阻碍模板

> 适用：与主角形成长期对抗或限制关系的角色（如老韩之于刘一、陈婆的限制功能）。
> 目标：确保阻碍者不是"为反而反"的工具人，有独立的欲望和恐惧。

### 阻碍三环

```
            ┌──────────────┐
            │  外层阻碍     │ ← 角色对主角做了什么？（具体行为）
            │  "我拦你"    │
            └──────┬───────┘
                   │
         ┌─────────┴─────────┐
         │   中层驱动        │ ← 角色为什么这么做？（核心需求）
         │   "我要……"       │
         └─────────┬─────────┘
                   │
         ┌─────────┴─────────┐
         │   内层恐惧        │ ← 角色在怕什么？（不敢面对的）
         │   "我怕……"       │
         └───────────────────┘
```

### 示例：老韩

| 环 | 内容 |
|----|------|
| **外层阻碍** | 杀人后取走铜佛，三年后卖给刘一——让刘一陷入命案风险 |
| **中层驱动** | 贪财。杀人为了八万现金，取佛因为鎏金值钱，卖佛因为赌博欠债 |
| **内层恐惧** | 怕被抓。但更怕停下——一旦不赌了，就得面对石窟里那个人 |

### 示例：陈婆（作为限制者）

| 环 | 内容 |
|----|------|
| **外层阻碍** | 不接铜佛、不直接解释、不替刘一做决定——"东西不欠他，你也不欠他" |
| **中层驱动** | 见过太多。说多了没用，说少了误事。点到为止 |
| **内层恐惧** | 怕话说太满。有些事她也看不透——看不透的事不说，是七十年的规矩 |

### 校验

```
□ 阻碍者的三层是否全部填写？
□ 中层驱动是否独立于主角存在？（把主角拿掉，这个角色是否还有自己的人生？）
□ 内层恐惧是否具体可感？（不是"怕死""怕失败"这种空泛词）
```

## 模板二：主角心理锚点模板

> 适用：所有故事的主角（刘一）。
> 目标：确保主角的每个关键决策都有来自过去的锚点牵引，不是临时起意。

### 心理锚点三要素

```
过去某时刻的一个具体承诺/遗憾/亏欠
          ↓
      形成行为准则（比如"来路不明的不碰"）
          ↓
      在故事中被一个类似情境激活
          ↓
      驱使其做出与平时不同的关键决策（撬开底座/不退铜佛）
```

### 锚点设计规则

| 规则 | 说明 | 反例 |
|------|------|------|
| **一句话能说清** | 锚点必须是可简述的具体事件，不是抽象性格 | "他当兵时欠了一条命"→太模糊；"退伍时战友说'答应的事得做完'"→具体✓ |
| **与主线形成镜像** | 锚点情境和故事关键决策之间必须存在可辨识的对应关系 | 二十年前"答应的事没做完"→二十年后"铜佛在等一个人打开底座"→镜像=承诺兑现 |
| **只影响一次决策** | 锚点激活后推动一个关键行为，之后就退场。不拿来解释角色的一切 | 一个锚点解释所有行为→用力过猛 |
| **不独立成章** | 锚点以碎片形式渗透，不超过3句话。不会发展成独立闪回章节 | 大段军营回忆→喧宾夺主 |

### 示例：刘一的锚点

| 要素 | 内容 |
|------|------|
| **过去事件** | 退伍时战友说"答应的事，得做完"——记了二十年 |
| **行为准则** | 来路不明的不碰——碰了就得负责到底 |
| **激活情境** | 铜佛搁在架子上，下面压着一条命。陈国栋找上门。他要么退回去、要么打开 |
| **关键决策** | 撬开底座——"碰了，就得负责到底" |
| **呈现方式** | 马老三十述中插入一句话。不超过三句。不详写 |

### 校验

```
□ 主角锚点是否能用一句话说清？
□ 锚点情境和故事关键决策之间是否存在镜像关系？
□ 锚点是否只激活一次（而非反复拿来解释一切）？
□ 锚点呈现是否碎片化（≤3句话，不独立成章）？
```

## 模板三：反派善恶反差模板

> 适用：所有反派/灰色角色（老韩、同伙、甚至包括被抢劫的受害者陈国良）。
> 目标：确保反派不是纯恶工具人。每个反派至少有一个"可以被理解但不被原谅"的维度。

### 反差四象限

```
          善意维度（曾经/偶尔）
          │
    ④    │    ①
  伪善   │  真心帮过
  利用   │  但最终选择
  善意   │  了恶
  包装   │
          │
──────────┼──────────→ 恶意维度（核心/持续）
          │
    ③    │    ②
  纯粹   │  恶中有憾
  的恶   │  作恶时有
  无来由  │  片刻动摇
          │
```

### 设计要求

| 象限 | 位置 | 要求 |
|:----:|------|------|
| ① | 善意维度中+恶意维度中 | **最优位置**。角色有善的一面（帮过主角/有苦衷），但关键时刻选择恶。读者"理解但不原谅" |
| ② | 善意维度低+恶意维度高 | 可接受。恶行明确，但至少有一处"犹豫""动摇"的细节——不洗白，不美化，只证明他是人 |
| ③ | 善意维度低+恶意维度低 | ❌ 不允许。纯恶+无来由=工具人反派 |
| ④ | 善意维度高+恶意维度低 | ❌ 不允许。伪善包装=廉价反转 |

### 示例：老韩（象限②→①之间）

| 维度 | 体现 |
|------|------|
| **恶意核心** | 抢劫杀人。三年若无其事。被追查时反咬陈国栋敲诈勒索 |
| **善意碎片** | 上个月帮刘一挡工商检查——不是设局，是真心帮。帮了，也杀了人。两件事都真实 |
| **动摇瞬间** | 出门停半秒、手指甲扣封泥又放弃、在巷子里抱着铜佛站了好一会儿——三次犹豫，三次都被压回去 |
| **设计原则** | 善意碎片和恶意核心不互相抵消。帮过刘一≠可以杀人。两者的并存恰恰是"灰度"——好人也可能作恶，恶人也曾帮过人 |

### 示例：陈国良（受害者≠纯善）

| 维度 | 体现 |
|------|------|
| **受害事实** | 被抢劫、被杀、被藏尸三年——毋庸置疑的受害者 |
| **灰色维度** | 文物贩子。灰色交易。随身佛像藏交易记录防黑吃黑——他知道这一行的风险。他带着八万现金进石窟——那是1993年，他没有完全信任老韩，但贪念让他走进了那个石窟 |
| **设计原则** | 受害者的灰色不能成为"他活该"的暗示。只是在说——好人也会冒险，冒险的人也会死。这比纯善被戕害更接近真实 |

### 校验

```
□ 反派/灰色角色是否至少有一处"可以被理解"的维度？
□ 这个维度是否呈现为具体行为/细节（而非空泛的"他也有苦衷"）？
□ 善意碎片是否与恶意核心并存但不互相抵消？
□ 受害者是否有灰色维度（使其立体，但不暗示"活该"）？
```
