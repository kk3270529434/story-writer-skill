# 角色塑造模式与成长弧线 — "深夜小茶馆"风格

> 更新日志：2026-06-11 | v1.0 | 基于143篇短篇故事蒸馏

---

## 一、主角类型图谱

### 1.1 主角来源（按频率排序）
| 类型 | 占比 | 典型示例 |
|------|:----:|----------|
| "朋友的朋友"（转述型） | 40% | 听友大奎、老马、包子有肉的大姨夫 |
| 历史/民间真实人物 | 25% | 刘一手（圈河村）、王木匠 |
| 杨湃本人/身边人 | 20% | 老马（常来茶馆）、杨湃自己 |
| 虚构角色 | 15% | 各故事中的司机、学生、商人 |

### 1.2 职业分布
- **工薪/基层**：医生、教师、司机、工人、农民 → 80%
- **边缘人**：古玩贩子、算命先生、刑满释放者 → 10%
- **身份不明/闲散**：社会闲逛、干啥啥不成的 → 10%

> 核心原则：主角必须是**普通人**，读者能代入。不能是富豪/明星/高官。

### 1.3 性格弧光
| 类型 | 特征 | 示例角色 |
|------|------|----------|
| 从疑到信 | 一开始不信鬼神，经历后改变 | 大奎（科学工作者→亲历诈尸） |
| 从背到顺 | 命途多舛，因某事转运 | 老马（干啥赔啥→灰仙转运） |
| 从贪到悟 | 贪念惹祸，最终悔悟 | 贪便宜买二手冰柜的人 |
| 从怯到勇 | 胆小怕事，被迫挺身 | 花大夫（被吓→拿粮打僵尸） |
| 旁观者醒悟 | 目睹他人遭遇而反思 | 各种"我听说""我朋友"叙事中的"我" |

### 1.4 内心矛盾模式
- **理性 vs 灵异**：受过教育的人遭遇超自然（最高频）
  > "大奎人是科学工作者，怎么可能相信这种鬼神之事呢。"
- **贪欲 vs 良知**：面对诱惑时的挣扎
- **传统 vs 现代**：老一辈信，年轻人不信；或反过来
- **责任 vs 恐惧**：必须面对恐怖以保护他人

## 二、配角配置规律

### 2.1 标配配角卡
每篇故事通常配置以下角色（可兼任）：

| 角色功能 | 标签 | 作用 |
|----------|------|------|
| **信息源** | "一个朋友跟我说" | 提供故事的原始素材 |
| **怀疑者** | 不信鬼神的理性派 | 制造张力，代表读者视角 |
| **知情老人** | "村里的老人""建国他爷爷" | 提供民俗解释/历史背景 |
| **喜剧担当** | 说话逗的角色 | 调节气氛，在恐怖后给笑点 |
| **受害者/目击者** | 亲历恐怖的人 | 推动情节 |

### 2.2 配角数量
- 短篇（单集）：3-5个有名有姓的角色
- 中篇（上下集）：5-8个角色
- 长篇（3集+）：8-15个角色

### 2.3 命名习惯
- 用**姓 + 称呼**：大奎、老马、花大夫、刘大姐、王大爷
- 极少给全名（全名仅用于历史事件）
- 功能性命名：包子有肉（网名）、老姨、二舅、建国他爷

## 三、成长弧线轨迹

### 3.1 篇幅与弧线关系
| 篇幅 | 完整弧线的角色数 | 弧线深度 |
|------|:----------------:|:--------:|
| 单集短篇 | 1个（主角） | 浅弧：一个转折点即可 |
| 上下集 | 1-2个 | 中弧：2-3个转折 |
| 长篇连载 | 2-3个 | 深弧：完整的起承转合 |

### 3.2 弧线触发点模式
1. **物品触发**：一件古董/二手车/旧冰柜 → 引出背后故事 → 人物改变
2. **地点触发**：进入凶宅/鬼楼/荒村 → 遭遇灵异 → 认知改变
3. **时间触发**：某个特定日期（清明、七月半、本命年）→ 事件发生 → 命运改变
4. **人际关系触发**：结识某人/得罪某人 → 因果报应 → 态度改变

### 3.3 弧线标志性关键词
| 弧线类型 | 开头关键词 | 转折关键词 | 结尾关键词 |
|----------|------------|------------|------------|
| 救赎弧 | 不信、倒霉、贪念 | 遇见、发现、原来 | 明白、改过、信了 |
| 堕落弧 | 顺遂、傲慢、不信邪 | 触犯、惹上、没想到 | 悔之晚矣、报应 |
| 顿悟弧 | 困惑、为什么我这么背 | 才知道、敢情是 | 从此、打那往后 |
| 归来弧 | 离开、多年后、重回 | 变了、不一样了 | 物是人非 |

## 四、角色塑造禁忌

### ❌ 禁止：
- 完美主角（必须有小缺点或偏见）
- 纯恶反派（即便是鬼/怪也有缘由）
- 角色道德说教（让情节暗示道德，不让角色宣讲）
- 女性角色花瓶化（虽然年代设定可能导致传统角色，但要有能动性）
- 无来历的"高人"/"大师"（必须解释其能力来源）

### ✅ 必须：
- 每个有名有姓的角色至少有一处个性化语言/行为
- 主角在故事中至少经历一次"信与不信"的摇摆
- 至少一个角色代表"普通人的反应"（害怕、逃跑、求助）
- 鬼怪有"人性化动机"——即便是害人，也讲原因

## 五、真实反应与配角独立性（v2.3 新增）

### 5.1 真实反应原则（P0）

**核心**：问自己——"如果是我亲身遇到这件事，我会怎么做？"把那个真实的、可能有点"怂"的反应写出来。

**"第一次接触异常"的完整反应链**：
```
否认（"不可能，电话没插线"）
  ↓ 至少1段
困惑（"但它确实响了"）
  ↓ 至少1段
求证（检查插头、问别人、白天再去看）
  ↓ 至少1段
害怕（"那我接的那个电话到底是什么"）
  ↓ 至少1段
再次接触或被迫面对
```

> 反例：刘波凌晨接死电话→犹豫"十秒"→接听。🔴 反应链被压缩到不真实。

**"秘密武器化"不可过快**：
```
第一次接触：恐惧 + 回避
第二次应验：困惑 + 半信半疑
第三次应验：好奇 + 试探性依赖
第四次以上：才可能产生"这是我的秘密武器"的贪念
```
> 反例：第一次预言应验→立刻把电话当"好运器"。🔴

**情绪倒退允许**：
- 可以在"不怕了"之后又被吓到，再退回"害怕"——真实反应不是线性的
- "我已经不怕了"→ 下一段被吓得更厉害 = 人类正常反应

### 5.2 配角独立性（P1）

**核心**：配角不能只是"信息播音员"——在主角需要时出现，说完信息就消失。

**配角独立性的三个强制要求**：

1. **有自己的欲望**：宿管大爷不只是"提供信息的NPC"——他为什么还在这工作？他怕不怕这部电话？他有没有想离开但离不开的原因？

2. **至少拒绝一次**：配角被问到时不能有问必答。至少有一次：
   - 转移话题（"你问这干嘛"）
   - 给出不完整/不准确的信息
   - 直接拒绝（"别问了"）
   
3. **有缺点**：
   - 宿管大爷：或许喝酒误事、或许胆小不敢管、或许曾经说谎掩盖了什么
   - 信息源角色：记忆模糊、添油加醋、有自己的偏见

> 反例：宿管大爷问什么答什么——电话谁打的、几时出的事、其他人什么下场，全部和盘托出。🔴 工具人。

**配角自检**：
- [ ] 这个配角有自己的欲望/顾虑吗？（与主线无关的）
- [ ] 他/她至少拒绝过一次主角的提问吗？
- [ ] 他/她给出的信息可能有误吗？
- [ ] 删掉这个配角，主角能自己获取这些信息吗？（如果能——配角是多余的）
