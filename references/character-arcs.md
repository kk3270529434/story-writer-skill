# 角色塑造模式与成长弧线 — "深夜小茶馆"风格

> 更新日志：2026-06-21 | v2.7 | 第二轮优化：核心角色灰度法则、配角反转功能、不可靠叙述设计

---

## 一、主角三要素强制规则（P0 — 缺一不可）

> 所有故事主角必须同时满足以下三个要素。缺少任意一项 → 故事不成立，返回重设。

### 要素 1：表层诉求（具体可落地）

主角有一个**读者能在 3 秒内理解的具体目标**。不是抽象的"想过好日子"，而是可执行的行动。

| ❌ 空泛 | ✅ 具体可落地 |
|--------|-------------|
| "想要转运" | "想把欠的三万块钱还上" |
| "想过好日子" | "想保住这个月的工资不被扣" |
| "寻找真相" | "找到失踪三天的父亲" |
| "想变强" | "治好母亲的病" |

**原则**：这个诉求应该能用一句话说清，且读者能立刻判断"他做到了吗"。

### 要素 2：隐藏心结（与结局呼应）

主角有一段**过往的遗憾、愧疚或秘密**，被故事中的事件重新激活。结局必须与这个心结形成呼应。

| 心结类型 | 示例 | 结局呼应 |
|----------|------|----------|
| 未尽的愧疚 | 年轻时为自保没有救一个落水同伴 → | 多年后在同一个水域面对类似选择 |
| 被压抑的恐惧 | 小时候见过一次"不该看见的东西" → | 成年后同类事件再次发生 |
| 家族秘密 | 爷爷死前留了一句话没说完 → | 故事的真相补全了那句话 |
| 未兑现的承诺 | 答应过某人但没做到 → | 结局用另一种方式完成承诺 |

**原则**：心结不需要在第一段就揭示——可以在 #02 或 #03 通过回忆/对话自然带出。

### 要素 3：失败代价（具象化的损失）

如果主角失败，**损失必须是读者能看见、能感受到的具体事物**。禁止空泛的"会死""世界毁灭"。

| ❌ 空泛代价 | ✅ 具象代价 |
|-----------|-----------|
| "会死" | "家里最后一个知道他下落的人也会消失" |
| "世界毁灭" | "他女儿会在下个月满七岁时——和前面六个孩子一样" |
| "失去一切" | "祖宅会被拆掉，爷爷的坟也会被迁走" |
| "受到惩罚" | "两只手会像他父亲一样——慢慢变黑，然后废掉" |

**代价三要素**：
- **可见**：读者能想象出那个画面
- **不可逆**：失去的不能再回来
- **与故事规则相关**：不是随机的惩罚

### 主角自检

```
□ 表层诉求：一句话能说清主角想要什么吗？
□ 隐藏心结：有没有一段未尽的过往被故事重新激活？
□ 失败代价：如果主角失败，读者能看到什么具体的损失？
```

## 二、主角类型图谱

### 2.1 来源分布
| 类型 | 占比 | 典型示例 |
|------|:----:|----------|
| "朋友的朋友"（转述型） | 40% | 听友大奎、老马、包子有肉的大姨夫 |
| 历史/民间真实人物 | 25% | 刘一手（圈河村）、王木匠 |
| 杨湃本人/身边人 | 20% | 老马（常来茶馆）、杨湃自己 |
| 虚构角色 | 15% | 各故事中的司机、学生、商人 |

### 2.2 职业分布
- 工薪/基层（80%）：医生、教师、司机、工人、农民
- 边缘人（10%）：古玩贩子、算命先生、刑满释放者
- 身份不明/闲散（10%）

> **核心原则**：主角必须是普通人，读者能代入。不能是富豪/明星/高官。

### 2.3 性格弧光
| 类型 | 特征 | 示例 |
|------|------|------|
| 从疑到信 | 一开始不信鬼神，经历后改变 | 大奎（科学工作者→亲历诈尸） |
| 从背到顺 | 命途多舛，因某事转运 | 老马（干啥赔啥→灰仙转运） |
| 从贪到悟 | 贪念惹祸，最终悔悟 | 贪便宜买二手冰柜的人 |
| 从怯到勇 | 胆小怕事，被迫挺身 | 花大夫（被吓→挺身对抗） |
| 旁观者醒悟 | 目睹他人遭遇而反思 | 转述型故事中的"我" |

## 三、配角功能定位规则（P1 — 强制）

> 每个出场配角必须承担至少 1 项剧情功能。禁止"信息播音员"式工具人——在主角需要时出现，说完信息就消失。

### 3.1 四种合法功能

| 功能 | 说明 | 示例 |
|------|------|------|
| **推动线索** | 提供关键信息，但信息获取有阻力 | 宿管大爷知道电话来历但不愿说 |
| **调节情绪** | 在恐怖处提供缓冲（搞笑/吐槽/理性质疑） | 不信鬼的朋友陪主角去探鬼楼，一路吐槽 |
| **反衬主角** | 与主角形成对比，凸显主角特质 | 同时遇鬼，配角崩溃而主角勉强撑住 |
| **埋下伏笔** | 配角的某句话/某个行为在后期成为关键 | 开头一句无心的话 → 结尾揭示是真相的钥匙 |

### 3.2 配角独立性三要求

1. **有自己的欲望**（与主线无关）：宿管大爷不只是"提供信息的NPC"——他为什么还在这工作？他怕不怕这部电话？
2. **至少拒绝一次**：配角被问到时不能有问必答。至少转移话题/给出不完整信息/直接拒绝各一次。
3. **可能有误**：提供信息的人可以记错、添油加醋、有偏见——信息不一定是 100% 准确的。

### 3.3 配角数量
- 短篇（单集）：3-5 个有名有姓的角色
- 中篇（上下集）：5-8 个角色
- 长篇（3集+）：8-15 个角色

### 3.4 命名习惯
- 姓 + 称呼：大奎、老马、花大夫、刘大姐、王大爷
- 极少全名（仅用于历史事件）
- 功能性命名：包子有肉（网名）、老姨、二舅

## 四、怪异/反派动机规则（P0 — 强制）

> 禁止"无缘无故害人"的设定。所有超自然存在必须有因果逻辑。

### 4.1 四种合法动机

| 动机类型 | 逻辑 | 示例 |
|----------|------|------|
| **生前执念** | 死前有未完成的事，死后仍在继续 | 被冤枉处决的犯人，死后还在找能证明他清白的人 |
| **被冒犯报复** | 活人触犯了某个地界/规矩，遭到了反击 | 施工队挖断了古墓的封土，全队开始做同一个噩梦 |
| **遵守地界规则** | 它不是在害人——是在执行某个超出人类理解的地界规则 | 黄河每年收三个人，不是针对谁，是规矩 |
| **寻求帮助** | 怪异现象其实是某种求助信号 | 鬼魂反复展示某个场景，其实是在提示真凶 |

### 4.2 动机自检

```
□ 怪异/反派为什么要在这个时间、对这个主角产生影响？
□ 它的行为逻辑全文一致吗？（不能前半段无差别害人，后半段突然有选择性）
□ 如果没有主角——它还会存在吗？（不会 → 它是"为了故事才被创造出来的"，需要重设）
```

## 五、真实反应原则（P0）

### 5.1 "第一次接触异常"完整反应链

```
否认（"不可能，电话没插线"）
  ↓ 至少1段描写
困惑（"但它确实响了"）
  ↓ 至少1段描写
求证（检查插头、白天再去看、问别人）
  ↓ 至少1段描写
害怕（"那我接的那个电话到底是什么"）
  ↓ 至少1段描写
再次接触或被迫面对
```

> **禁止跳过中间情绪**。反例：凌晨接死电话→犹豫"十秒"→接听。反应链被压缩到不真实。

### 5.2 "秘密武器化"不可过快

```
第一次接触：恐惧 + 回避
第二次应验：困惑 + 半信半疑
第三次应验：好奇 + 试探性依赖
第四次以上：才可能出现"这是我的秘密武器"的贪念
```

> 反例：第一次预言应验 → 立刻把电话当"好运器"。违规。

### 5.3 情绪倒退允许

- 可以在"不怕了"之后又被吓到，退回"害怕"——真实反应不是线性的
- "我已经不怕了"→下一段被吓得更厉害 = 人类正常反应

## 六、角色塑造禁忌

### ❌ 禁止
- 完美主角（必须有小缺点或偏见）
- 纯恶反派（即便是鬼/怪也必须有缘由）
- 角色道德说教（让情节暗示，不让角色宣讲）
- 女性角色花瓶化（年代设定可传统但有能动性）
- 无来历的高人/大师（必须解释能力来源）
- 角色说出超越其教育/年龄/时代背景的话

### ✅ 必须
- 每个有名有姓的角色至少一处个性化语言/行为
- 主角至少经历一次"信与不信"的摇摆
- 至少一个角色代表"普通人的反应"（害怕、逃跑、求助）

## 七、核心角色灰度法则（P0 — 新增）

> **所有主要角色必须同时具备"光明动机"与"黑暗隐秘"。禁止任何角色非黑即白。**

| 角色类型 | 光明动机（读者共情） | 黑暗隐秘（制造灰度） | 如何服务于反转 |
|----------|-------------------|---------------------|--------------|
| **主角** | 善良、想帮忙、有正义感 | 有自己的执念/私心/隐瞒，其行为会影响剧情走向 | 主角的"善良帮助"可能是帮了倒忙 |
| **事件核心人物** | 可怜、值得同情、有苦衷 | 有自我欺骗/刻意隐瞒/间接作恶的一面 | 其"苦衷"可能是自我美化的谎言 |
| **灵异主体** | 有执念可理解、让人怜悯 | 有怨念、私心甚至过错，执念不全是善意 | 其"求助"可能是复仇的伪装 |

**自检**：
```
□ 主角：除了"想帮忙"之外，他有什么私心？（怕丢工作/曾经亏欠某人/在逃避什么）
□ 事件核心人物：他的"苦衷"经得起推敲吗？有没有他故意不说的事？
□ 灵异主体：它真的只是想"传个话"吗？如果它另有所图呢？
```

## 八、配角功能升级规则（P1 — 新增）

> **禁止纯工具人配角。每个出场配角必须由"信息源"升级为"剧情参与者"。**

### 8.1 配角三升级

| 原功能 | 升级后功能 | 示例 |
|--------|----------|------|
| 提供信息的 NPC | **误导者**：给出的信息不完整/有偏/故意错 | 刘大姐说"老孙头怕别的东西"——她在隐瞒自己知道的事 |
| 气氛调节的工具 | **伏笔承载者**：一句话/一个行为在后面成为关键 | 小吴说走就走——走之前他做了什么？ |
| 路过/消失的群演 | **反转参与者**：在故事后半段揭晓与核心事件的关联 | 老孙头不是腿疼走的——他认识范秀兰 |

### 8.2 配角反转规则
至少 1 个配角，在故事后半段（#03-#04）揭晓其与核心事件的**直接关联**（目击者、参与者、受益者、知情不报者）。

## 九、不可靠叙述角色设计规范（P2 — 新增）

> **可选择 1 名核心角色作为不可靠叙述者，利用记忆偏差/刻意隐瞒制造反转。**

### 9.1 不可靠叙述的四种模式

| 模式 | 叙述者特点 | 读者何时发现 | 适用反转类型 |
|------|----------|------------|------------|
| **自我美化** | 把自己说得比实际更好/更无辜 | #03-#04，通过物证矛盾揭露 | 身份反转 |
| **选择性遗忘** | 刻意跳过某些关键记忆 | #03，通过灵异事件还原被遗忘的片段 | 记忆偏差 |
| **谎言嵌套** | 叙述中夹杂着编造的细节 | #04，通过他人视角戳穿 | 人为+灵异叠加 |
| **无知叙述** | 叙述者自己也不知道真相 | #04，真相揭晓时叙述者和读者同时震惊 | 因果倒置 |

### 9.2 不可靠叙述的使用信号
叙述者的陈述中出现以下矛盾时，就是在暗示"不可靠"：
- 时间线对不上（"我记得是1983年"——但物证显示1984年）
- 细节过于完美（回忆中场景精确到每句话——太完整反而可疑）
- 回避关键问题（每次问到核心就转移话题）
- 多个角色的描述互相矛盾
