# 情节结构规律 — "深夜小茶馆"风格

> 更新日志：2026-06-21 | v2.7 | 第二轮优化：双重真相叙事结构、伪解答三层递进、5类反套路骨架、伏笔强制规则

---

## 一、强制悬念密度规则（P0 硬指标）

> **核心原则**：故事必须像过山车——上去、下来、急转弯、再上去。不允许平铺直叙。

按篇幅字数设置以下不可协商的硬指标：

| 位置 | 指标 | 判定标准 |
|------|------|----------|
| **开篇 300 字内** | 出现第一个反常细节 | 必须有一个让读者"嗯？"的瞬间。可以是物品异常、人物异常、环境异常、对话异常。纯背景介绍/人物铺垫/年代说明 ≥300 字而未抛钩 → 违规 |
| **每 800 字** | 出现 1 次小转折/小危机 | 信息被颠覆、情况变糟、出现新疑点、角色情绪明显变化。禁止连续 800 字无冲突推进的"平铺区" |
| **每 1500 字** | 出现 1 次中级高潮 | 主角直面危险、被迫做出选择、或发现关键线索。紧张感明显拉升 |
| **结尾前 300 字** | 必须有反转/留白/因果落点 | 不可平淡收尾。反转必须"出人意料但合情合理" |

**自检方法**：写完后用字数统计定位上述节点，逐点标注是否达标。未达标位置 → 重写对应段落。

## 二、3幕6节点强制骨架

> 替代原"三幕式"模糊结构。每个节点有明确的叙事任务、悬念设计和情绪等级。

### 2.1 骨架总览

```
┌──────────────────────────────────────────────────────────┐
│  [外层框架] 杨湃茶馆叙事（开场白 + 收场对）               │
│  ┌────────────────────────────────────────────────────┐  │
│  │ [内层故事] 3幕6节点                                 │  │
│  │                                                    │  │
│  │ 第一幕·铺垫              第二幕·发展               │  │
│  │ 节点1 → 节点2            节点3 → 节点4              │  │
│  │ 钩子抛出 → 危机升级      求助受阻 → 终极危机        │  │
│  │                                                    │  │
│  │ 第三幕·收束                                        │  │
│  │ 节点5 → 节点6                                       │  │
│  │ 真相反转 → 收尾余味                                 │  │
│  └────────────────────────────────────────────────────┘  │
└──────────────────────────────────────────────────────────┘
```

### 2.2 各节点详细定义

#### 第一幕：铺垫

| | 节点1：钩子抛出 | 节点2：危机升级 |
|------|------|------|
| **位置** | #01 开篇 | #01 尾 → #02 头 |
| **叙事任务** | 时空锚定 + 主角登场 + 首次接触反常 | 反常加剧 + 开始影响主角的日常生活 |
| **悬念类型** | "不完全信息"钩子 — 展示异常但不解释 | "这不对劲"钩子 — 读者感知到危险在升级 |
| **情绪等级** | 轻度恐怖 (2-3/10) | 轻度→中度过渡 (3-5/10) |
| **输出标准** | 读者知道"谁 + 在哪 + 什么时候 + 什么不对劲" | 主角无法再忽视异常；被迫采取第一个行动 |
| **禁止** | ❌ 在此节点就揭示真相 | ❌ 主角反应过于平淡；❌ 异常反而被"正常化" |

#### 第二幕：发展

| | 节点3：求助受阻 | 节点4：终极危机 |
|------|------|------|
| **位置** | #02 中段 | #03 |
| **叙事任务** | 主角试图弄清真相，但每一条路都有阻力 | 生命受到直接威胁，退路被切断 |
| **悬念类型** | "谁在隐瞒"钩子 — 信息源的不可靠制造悬疑 | "冲向悬崖"钩子 — 危险升级到无法回头 |
| **情绪等级** | 中度恐怖 (5-7/10) | 重度恐怖 (7-9/10) |
| **输出标准** | 主角至少经历 1 次失败的求证 + 1 次被拒绝/被误导 | 主角面临"我可能过不去了"的绝望瞬间 |
| **禁止** | ❌ 知情人有问必答；❌ 查资料一帆风顺 | ❌ 危机被一句话解决；❌ 主角全程冷静 |

#### 第三幕：收束

| | 节点5：真相反转 | 节点6：收尾余味 |
|------|------|------|
| **位置** | #04 | END 之前 |
| **叙事任务** | 揭示核心真相，方向出乎意料 | 后果落地 + 给读者留下最后一个念想 |
| **悬念类型** | "原来如此"大揭示 — 但必须是升级而非降级 | "那后来呢"余味 — 不把话说尽 |
| **情绪等级** | 高峰后急速反差 (9→5/10) | 回落到温暖/哲理/细思极恐 (3-4/10) |
| **输出标准** | 揭示同时展示代价；反转让读者觉得"赚了" | 最后一段是具体画面而非空洞金句 |
| **禁止** | ❌ 反转是降级；❌ 用对话解释全部真相；❌ "交给警察""一切恢复正常" | ❌ AI 味金句总结；❌ 所有伏笔全部收回 |

### 2.3 单集 vs 多集适配

| | 单集故事 | 多集故事 |
|------|------|------|
| 章节数 | #01~#04（4节） | 每集 #01~#04 |
| 6节点映射 | 6个节点分布在4节内 | 上集走节点1-3，下集走节点4-6 |
| 断点位置 | 无断点 | 在节点3（求助受阻）之后断 |
| 下集开头 | — | 2-3句简短回顾 + 直接续，不重复上集 |

## 三、冲突逐级升级法则

> 矛盾必须从"疑似反常"到"影响生活"再到"生命威胁"逐级放大。禁止跳级或后期弱化。

### 3.1 升级阶梯

```
Level 1: 疑似反常（#01）
  └→ 主角注意到一件无法解释的小事
  └→ 反应：否定、找合理化解释
  └→ 例：电话没插线却响了、鸡全部面朝一个方向

      ↓ 升级触发：反常再次发生/应验

Level 2: 影响生活（#01尾 - #02）
  └→ 反常开始干扰主角的日常生活
  └→ 反应：困惑、半信半疑、开始打听
  └→ 例：电话预言成真、面壁的鸡当晚遭了灾

      ↓ 升级触发：求证受阻/遭遇更严重的直接威胁

Level 3: 人身威胁（#02尾 - #03）
  └→ 主角自身安全受到明确威胁
  └→ 反应：恐惧、逃跑/求助、第一次尝试对抗
  └→ 例：电话宣告了主角的名字和死亡时间

      ↓ 升级触发：发现背后更大的真相/代价

Level 4: 终极危机（#03尾 - #04）
  └→ 退路被切断，必须直面
  └→ 反应：绝望→拼死一搏
  └→ 例：主角必须在死亡日期到来前找到破解方法
```

### 3.2 升级自检

- [ ] Level 1 → Level 2：有没有一个"反常再次发生/应验"的触发事件？
- [ ] Level 2 → Level 3：威胁有没有从"影响生活"升级到"人身安全"？
- [ ] Level 3 → Level 4：主角有没有"无路可退"的绝望时刻？
- [ ] 全程有没有 Level N+1 的恐怖弱于 Level N 的情况？（有→违规）

## 四、节尾钩子规则（强制）

每个 `#01`、`#02`、`#03` 的结尾必须留钩子：

| 节 | 钩子类型 | 操作 | 示例 |
|----|----------|------|------|
| #01 尾 | **不完全信息钩子** | 展示异常现象但不解释 | "刘波看着那部电话——依然没插线，插头垂在墙上晃来晃去。" |
| #02 尾 | **假安全感钩子** | 看似平息但读者感觉不对劲 | "电话里的预言一个接一个地应验。刘波心想：这东西，好像不是坏东西。" |
| #03 尾 | **冲向悬崖钩子** | 危险升级到最高，角色做出决定 | "电话那头的声音说——'一月十三号。凌晨两点十三分。419。刘波。'" |

## 五、年代考据剧情化规则

> 所有年代元素（歌曲、物件、社会背景）必须成为剧情的一部分。禁止单独堆砌作为背景板。

### 5.1 年代元素剧情化三路径

| 路径 | 操作 | 示例 |
|------|------|------|
| **怪事载体** | 年代物件本身就是灵异现象的载体 | 一台 90 年代的 BB 机收到来自 3 年前已死之人的传呼 |
| **线索道具** | 年代物品是破解谜团的关键道具 | 一张 1983 年的旧粮票背面写着一个人的生辰八字 |
| **冲突诱因** | 年代背景直接导致或加剧了冲突 | 90 年代国企下岗潮中，失业工人为了生计触犯了某条禁忌 |

### 5.2 呈现方式

- ✅ 通过角色行为呈现："大奎掏出 BB 机看了一眼——空了。这月的工资全他妈随了份子。"
- ✅ 通过叙述者批讲："好多年轻朋友不知道，那会儿打电话得去邮局，半夜哪有手机这玩意。"
- ❌ 禁止："1983 年的流行歌曲有《乡恋》《我的中国心》《在希望的田野上》。"（百科式罗列）

### 5.3 年代元素自检

- [ ] 每个年代元素是否都标注了对应的剧情节点？
- [ ] 有没有任何年代元素仅出现在背景描述中而不参与剧情？（有→删除或剧情化）
- [ ] 年代元素数量：每篇 2-3 个为宜，不超过 5 个

## 七、情节禁忌

### ❌ 禁止
- 纯恐怖无缓冲（最恐怖场景后必须有缓冲段落）
- 真相完全依靠外部权威揭示（至少一处是角色自己发现的）
- 所有伏笔全部收束（留 1-2 处"未解之谜"）
- 角色在没有任何铺垫的情况下突然勇敢/聪明
- 鬼怪/怪异没有"来由"（必须给一个解释框架）
- 超过 800 字无冲突推进的段落
- 用初级方式解决终极危机（如"说声谢谢就解除诅咒"）
- 结尾用"交给警察""一切恢复正常"消解恐怖

### ✅ 必须
- 每篇至少一个"细思极恐"的细节
- 至少一处"原来如此"的揭示（反转升级）
- 单集故事结尾提供情感满足或余味
- 主角解决危机至少经历 3 步：尝试→失败→再试

## 八、双重真相叙事结构（P0 — 叠加在3幕6节点之上）

> **核心**：在原有"3幕6节点"骨架上，叠加"真相层级"。每一幕对应一个真相层级。

| 幕 | 3幕6节点 | 真相层级 | 读者的认知状态 |
|----|----------|----------|--------------|
| **第一幕** | 节点1→2 | **表层假象** | 读者形成俗套预判："这应该是XXX的故事" |
| **第二幕** | 节点3→4 | **伪解答** | 给出看似合理的真相，读者以为接近答案——但有细心的读者会感到不对劲 |
| **第三幕** | 节点5→6 | **深层真相+落点** | 推翻伪解答，揭示人性+灵异+时代的复杂答案 |

### 8.1 伪解答三层递进规则

```
第一重伪解答（#02中段）：
  最符合俗套认知的答案
  由主角/配角主动提出
  读者感受："哦，原来是这样"
  └→ 但埋下1处矛盾伏笔

第二重伪解答（#02尾-#03头，可选）：
  略带反转的答案
  由剧情线索引导读者自行推断
  读者感受："等等，好像不止是这样"
  └→ 但仍然不完整，再埋1处矛盾伏笔

终极真相（#03尾-#04）：
  人性+灵异结合的复杂答案
  同时解释前文所有反常细节（包括前两重伪解答的矛盾之处）
  读者感受："天哪，原来是这样——回头想想全说得通了"
```

### 8.2 双重真相自检
- [ ] 读者能否在读完后，回头发现至少 3 处"原来那时就在暗示"的细节？
- [ ] 伪解答是否"合理但不完整"（而非"明显错误"）？
- [ ] 终极真相是否比伪解答更惊人（升级而非降级）？
- [ ] 前文所有主要反常细节是否都在终极真相中得到解释？

## 九、伏笔强制规则（P0）

> **每个反转点，必须在对应章节前埋下至少 3 处伏笔。全篇总计至少 6 处伏笔。**

### 9.1 三类伏笔

| 类型 | 形式 | 如何让读者"视而不见" |
|------|------|-------------------|
| **物品异常** | 特定物品出现/位置/状态异常 | 放在一堆正常描述中，一笔带过 |
| **人物反应反常** | 某人对某事的反应过度/不足 | 用角色的性格/习惯合理化——"他平时就这样" |
| **对话歧义** | 一句话有双重解读 | 当时的语境让读者自然地选择"正常"解读 |

### 9.2 伏笔埋设位置要求

| 反转点 | 对应伏笔 | 位置要求 |
|--------|:--------:|----------|
| 第一层反转（推翻表层假象） | 伏笔1、2、3 | 分别在 #01、#02头、#02中 |
| 第二层反转（推翻伪解答） | 伏笔4、5、6 | 分别在 #02尾、#03头、#03中 |

### 9.3 伏笔自检
- [ ] 每处伏笔在"伏笔位置"读起来是否完全合理（不被读者立刻识别为伏笔）？
- [ ] 每处伏笔在"回收位置"是否让读者产生"原来如此"的感受？
- [ ] 有没有"一次性伏笔"——出现了但从未被回收？（有→删除或回收）

## 十、5类反套路剧情骨架（强制至少选用1类）

> 创作时从以下骨架中选取至少 1 类作为故事结构基底。搭配 `assets/anti-cliche.md` 使用。

### 骨架 1：身份反转类
**核心**：受害者→加害者，求助者→复仇者，旁观者→始作俑者

**结构**：
```
#01-#02：建立 A 身份（可怜的寻亲者/无辜的受害者）
#03：发现第一重矛盾——A 的说法与物证对不上
#04：揭晓——A 不是受害者，A 是加害者（或A在自我欺骗）
```
**代表反转**："老范找妈" → 老范知道妈在哪，他在找的是妈藏起来的东西

### 骨架 2：时空错位类
**核心**：灵异不是当下的亡魂，是过去/未来的时空信号，主角行为会影响过去

**结构**：
```
#01-#02：主角接收到"灵异信号"（电话/收音机/旧信件）
#03：发现信号来自过去某个时间点
#04：主角的回应改变了当年的某个选择——但带来了新的后果
```
**代表反转**：主角接的电话来自 1968 年，他说的一句话改变了范秀兰的决定

### 骨架 3：人为+灵异叠加类
**核心**：有人利用传说装神弄鬼掩盖罪行，但真的有灵异存在，且两者目标不同

**结构**：
```
#01-#02：灵异事件发生，线索指向"闹鬼"
#03：发现有人为操纵的痕迹——"原来是人在搞鬼"
#04：人为的真相之下——还有一层灵异的真相，且两者的"目标"是对立的
```
**代表反转**：工头在装鬼吓人（为了掩盖偷工减料），但工地真的有鬼——鬼在试图暴露工头的罪行

### 骨架 4：记忆偏差类
**核心**：主角/核心人物的记忆被篡改或自我美化，灵异事件在还原真实记忆

**结构**：
```
#01-#02：主角回忆过去，引出灵异事件
#03：灵异事件的细节与主角的回忆产生矛盾
#04：揭晓——主角的记忆是被自己篡改过的，灵异事件在还原真相
```
**代表反转**："老范记得妈每天接他放学"→真相是老范的妈根本没接过他——那是他幻想的。电话里的声音问他"小宝放学没有"——是妈在找他，而不是他在找妈

### 骨架 5：因果倒置类
**核心**：不是亡魂等了主角几十年，是主角的到来唤醒/造就了当年的悲剧

**结构**：
```
#01-#02：主角遇到灵异事件，看似是"被找上门"
#03：发现灵异事件与主角的个人历史有关
#04：揭晓——主角当年的某个行为，间接导致了悲剧的发生
```
**代表反转**：二勇发现——电话里的老太太认识他。二十六年前，范秀兰死前最后一通电话，转接错了——转接到了二勇他爸的单位。二勇他爸是当年的电话接线员。那通没接通的电话，改变了两个家庭的命运
