# 灵异现象处理规则

> 更新日志：2026-06-26 | v1.0 | 新建：现实解释+一处留白双轨输出，禁用强行祛魅

---

## 使用说明

故事中出现的所有"灵异现象"，必须遵循双轨处理规则。在**步骤二（情节骨架）**阶段标注每个灵异现象的处理方式，在**步骤四（分步创作）**中按规则执行。

---

## 核心原则

```
故事中至少保留一处不可解释的灵异留白。
其他所有灵异现象都可以有现实解释。
两者的比例 ≈ N-1个可解释 : 1个不解释。
```

### 为什么必须保留一处留白

- 全解释 = 强行祛魅。读者感受："哦，全是巧合，没意思"
- 全不可解释 = 神棍故事。读者感受："编不下去了就说是鬼"
- N-1解释 + 1留白 = 最接近真实的灵异体验——你相信的那些，都可以不信。但有一件，你没法不信

## 一层：现实解释规则

### 解释方向库（四选一）

| 方向 | 说明 | 适用场景 | 示例 |
|------|------|----------|------|
| **物理现象** | 声音/温度/光线/震动有自然原因 | 物品异常 | 铜佛半夜嗡嗡响→座机振铃线圈共振 |
| **心理投射** | 恐惧/期待/愧疚放大了感官 | 体感异常 | 底座发热→人紧张手心潮，金属导热放大温差 |
| **人为干预** | 有人故意制造或利用了"灵异" | 事件异常 | 六次交易失败→陈国栋在背后拦截 |
| **环境巧合** | 天气/地质/生物导致的巧合 | 场景异常 | 风沙敲卷帘门→"像有人在窗外弹指甲" |

### 解释的呈现规则

| 规则 | 说明 | 反例 |
|------|------|------|
| **由角色揭示** | 解释通过角色之口说出（陈婆/刘一自己琢磨），不通过叙述者旁白 | "后来警方查明，嗡嗡声是因为……"→太像说明文 |
| **不占用大段篇幅** | 一句带过，不留恋。解释完了立刻切走 | 陈婆说座机共振→刘一愣了→切下一段。不停留 |
| **不解释情感** | 物理现象可以解释，情感体验不解释。嗡嗡声=线圈共振，但"半夜两点那通无人说话的电话"——不解释 | "他感到恐惧是因为童年阴影"→违规 |
| **允许角色不接受** | 角色可以继续怀疑、继续不安。物理解释了≠角色就释然了 | 刘一挪了座机铜佛不响了——但他每天放砖茶。理性的解释和感性的仪式可以共存 |

## 一层：不可解释留白规则

### 留白位置选择

| 位置 | 效果 | 示例 |
|------|------|------|
| **陈婆出场后** | 最自然——陈婆的"感觉"本身就是边界模糊的。她说出口的话可验证，没说的不可验证 | 陈婆说"在等人"。她怎么知道在等人？不解释 |
| **所有解释之后** | 最有力——读者以为全部合理了，忽然来一下解释不了的 | 座机拔了插头，铜佛仍自鸣一声——这就不是物理了 |
| **结尾收束处** | 最有余味——最后一件小事，不解释，翻篇 | 每天早上茶杯里的灰尘比别处细——"像碾碎的骨头末子" |

### 留白设计规则

| 规则 | 说明 | 反例 |
|------|------|------|
| **单次出现** | 一处留白，一次显现。不重复、不升级 | 铜佛自鸣了一次→够了。如果每夜都响→变成鬼故事 |
| **不推动剧情** | 留白是氛围，不是线索。不靠它破案、不靠它反转 | 铜佛自鸣没有揭示任何新信息。它就是响了一下——然后陈婆撒了落叶 |
| **不解释也不否认** | 叙述者不说是鬼，也不说是巧合。就让它悬着 | "可能是幻觉，也可能不是"→最差的写法。直接不说 |
| **物理上不可能但情感上合理** | 留白的"真实性"不来自物理逻辑，来自那一刻所有人都觉得它应该响 | 陈国栋在火车站等了三年→刘一撬开底座的那一刻→铜佛在空店里自鸣一声——物理上不可能，但没有人觉得它不该响 |

### 留白自检

```
□ 全篇是否恰有一处不可解释的灵异留白？（不是零处，也不是多处）
□ 这处留白是否发生在所有可解释现象被解释之后？
□ 是否不推动剧情、不揭示信息、不成为反转依据？
□ 叙述者是否克制——不解释也不否认？
□ 那一刻——读者是否觉得"它就该响"？
```

## 禁止项：强行祛魅

### 什么是强行祛魅

在故事结尾用一句话把前面所有灵异现象全部归因为巧合/幻觉/人为——不给任何一件留余地。

**反例**：
> "后来查明，所有怪事都是因为老房子地基下沉导致的管道共振，加上刘一长期失眠产生的幻觉。"

### 为什么禁止

- 消解了全篇积累的敬畏感
- 让灵异沦为廉价的"吓你一跳然后说哈哈骗你的"
- 西北民间叙事的底色是"有些事说不清"——强行说清=违背地域文化气质

### 允许的祛魅边界

- ✅ 多数现象有现实解释
- ✅ 保留恰一处不可解释
- ✅ 解释是角色自己发现的，不是叙述者盖棺定论的
- ❌ 用一句旁白/结语把所有灵异全部清零

## 西北灵异的文化基调

西北民间叙事对灵异的态度不是"信"或"不信"的二元对立。是"有些事——说不清"。

| 基调 | 体现 |
|------|------|
| **物老成精** | 老物件用久了会有"脾气"。铜佛不是鬼——但它替人守了三年证据，人欠它一个位置 |
| **地界有规矩** | 戈壁滩、废弃石窟——有些地方人进去要守规矩。不守就会出事。不是超自然惩罚，是自然规律被人格化了 |
| **人比鬼可怕** | 故事里最恐怖的东西从来不是灵异——是老韩若无其事活了三年。是陈国栋被反咬敲诈勒索 |
| **因果不虚但不直说** | 老韩三年卖不出铜佛——不是鬼挡财，是有人在挡。但"有人"之外，是不是还有别的？不回答 |
